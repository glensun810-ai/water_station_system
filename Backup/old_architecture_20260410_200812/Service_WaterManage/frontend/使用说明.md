## 💡 常见问题

### Q: 为什么直接打开 HTML 文件会显示空白？

A: 现代浏览器出于安全考虑，对 `file://` 协议有限制：
- 可能阻止加载本地 JavaScript 文件
- 可能阻止跨域 API 请求

**解决方案：** 使用 `index-standalone.html`（内嵌 Vue）或通过 HTTP 服务器访问。

### Q: 如何确认 Vue.js 是否加载成功？

A: 打开浏览器控制台（F12），输入：
```javascript
typeof Vue
```
如果返回 `"object"` 表示加载成功，`"undefined"` 表示失败。

### Q: Tailwind CSS 加载慢怎么办？

A: Tailwind CSS 使用 CDN，首次加载可能需要几秒。等待片刻或刷新页面。
