### 7.2 常见问题解决

**Q1: 端口被占用？**
```bash
# 查找占用进程
lsof -ti :8080

# 杀掉进程
kill -9 <PID>
```

**Q2: 404错误？**
```
确保使用正确的路径：/frontend/login.html
而不是：/water-admin/login.html
```

**Q3: 登录失败？**
```
检查用户名密码
查看控制台错误信息
检查数据库是否存在
```
