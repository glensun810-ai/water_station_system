# AI产业集群空间服务系统

## 🚀 快速启动

### 一键启动（推荐）

```bash
python start_services.py
```

启动后选择模式：
- **模式1**：新架构 - 统一API网关（推荐用于生产环境）
- **模式2**：临时模式 - 从Backup恢复旧架构（推荐用于快速启动）
- **模式3**：静态服务 - 仅前端页面（推荐用于预览）

## 🔧 启动模式对比

| 模式 | 适用场景 | 端口 | 功能完整度 |
|------|---------|------|-----------|
| 新架构 | 生产环境 | 8000 | ⚠️ 需完善配置 |
| 临时模式 | 快速启动 | 8008 | ✅ 完整可用 |
| 静态服务 | 前端预览 | 8000 | ❌ 无后端 |

## ⚡ 快速命令

```bash
# 查看运行中的服务
ps aux | grep python

# 查看端口占用
lsof -i :8000
lsof -i :8008

# 查看日志
tail -f logs/unified_api.log
tail -f logs/water_service.log

# 停止所有服务
pkill -f "python.*main.py"
```
