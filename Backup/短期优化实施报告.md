## 四、用户体验流程

### 用户管理流程
```
1. 超管登录 → Portal首页 → 系统管理 → 用户管理
2. 查看用户列表和统计数据
3. 搜索或筛选用户
4. 点击"编辑"修改用户信息
5. 分配办公室、设置角色
6. 保存修改
```

### 结算管理流程
```
1. 超管登录 → Portal首页 → 系统管理 → 结算管理
2. 查看结算概览（待结算、已申请、已结算）
3. 切换Tab查看不同状态记录
4. 选择记录进行批量操作
5. 申请结算或确认结算
```

### 新用户注册流程
```
1. 访问登录页面
2. 输入用户名和密码
3. 登录成功后，检测是否有办公室
4. 如果没有，显示办公室选择对话框
5. 选择办公室后确认
6. 跳转到Portal首页
```
