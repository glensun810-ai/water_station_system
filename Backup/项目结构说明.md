## 二、核心文件说明

### 2.1 设计系统（assets/css/design-system.css）

**用途**: 统一的CSS样式系统

**内容包括**:
- CSS变量系统（颜色、间距、圆角、字体）
- 统一按钮样式（.btn）
- 统一卡片样式（.card）
- 统一输入框样式（.input）
- 统一徽章样式（.badge）
- 工具类（Flex、Grid、间距、颜色等）

**使用方法**:
```html
<link rel="stylesheet" href="assets/css/design-system.css">
```

### 2.2 组件库

#### OfficeSelector - 办公室选择组件

**用途**: 统一的办公室选择界面

**使用示例**:
```html
<OfficeSelector 
  v-model="selectedOffice" 
  @change="handleOfficeChange" 
/>
```

**Props**:
- modelValue: 当前选中的办公室
- excludeInactive: 是否排除不活跃办公室（默认true）
- showLeader: 是否显示负责人（默认true）
- columns: 每行显示列数（默认2）

#### StatsCard - 统计卡片组件

**用途**: Dashboard统计卡片

**使用示例**:
```html
<StatsCard 
  title="今日预约" 
  :value="23" 
  icon="📅" 
  color="primary" 
  :trend="10" 
/>
```

**Props**:
- title: 统计标题
- value: 统计数值
- icon: 图标
- color: 颜色主题
- trend: 趋势值

#### ConfirmDialog - 确认对话框组件

**用途**: 统一的确认对话框

**使用示例**:
```html
<ConfirmDialog 
  v-model="showDialog" 
  title="确认删除" 
  message="删除后无法恢复" 
  type="danger" 
  @confirm="handleConfirm" 
/>
```

**Props**:
- modelValue: 是否显示
- title: 对话框标题
- message: 对话框消息
- type: 对话框类型（info/warning/danger）

#### FilterButton - 筛选按钮组件

**用途**: Tab切换和状态筛选

**使用示例**:
```html
<FilterButton 
  v-model="activeFilter" 
  :options="filterOptions" 
  type="segment" 
/>
```

**Props**:
- modelValue: 当前选中值
- options: 选项数组
- type: 按钮类型（segment/tab/card）

#### DatePicker - 日期选择器组件

**用途**: 统一的日期选择器

**使用示例**:
```html
<DatePicker 
  v-model="selectedDate" 
  :show-quick-select="true" 
/>
```

**Props**:
- modelValue: 选中的日期
- showQuickSelect: 是否显示快捷选择

#### OrderList - 订单列表组件

**用途**: 统一的订单列表查询

**使用示例**:
```html
<OrderList 
  :api-url="'/api/user/orders'" 
  :type="'water'" 
  @item-click="handleItemClick" 
/>
```

**Props**:
- apiUrl: API地址
- type: 订单类型（water/meeting）
- tabs: Tab配置

## 四、使用指南

### 4.1 开发新页面

1. 引入设计系统样式：
```html
<link rel="stylesheet" href="../assets/css/design-system.css">
```

2. 引入Vue 3：
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

3. 使用组件：
```html
<template>
  <OfficeSelector v-model="selectedOffice" />
  <StatsCard title="统计" :value="100" icon="📊" />
</template>
```

### 4.2 访问系统

## 五、维护指南

### 5.1 修改设计系统

编辑 `assets/css/design-system.css`

**注意事项**:
- 修改CSS变量会影响所有页面
- 保持向后兼容
- 测试移动端适配

### 5.2 新增组件

1. 在 `components/common/` 或 `components/business/` 创建新组件
2. 遵循Vue 3 Composition API规范
3. 编写完整的Props和Events定义
4. 添加组件使用文档

### 5.3 修改现有页面

- Portal首页: 编辑 `portal/index.html`
- 管理后台: 编辑 `portal/admin/index.html`
- 登录页面: 编辑 `portal/admin/login.html`
