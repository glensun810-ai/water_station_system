## 三、技术栈

- **前端框架**: Vue 3（Composition API）
- **样式系统**: CSS变量 + Tailwind CSS思想
- **响应式**: Flex + Grid + 媒体查询
- **移动端**: 安全区域适配
- **状态管理**: Vue ref/reactive
- **路由**: Hash路由
- **API**: fetch API
