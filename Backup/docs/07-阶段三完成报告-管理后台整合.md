### 3.2 权限管理系统

#### 角色定义

| 角色 | 权限范围 | 说明 |
|------|---------|------|
| **super_admin** | 所有服务 | 超级管理员，拥有所有权限 |
| **water_admin** | 水站管理 | 水站管理员，管理水站相关功能 |
| **meeting_admin** | 会议室管理 | 会议室管理员，管理会议室相关功能 |
| **finance_admin** | 财务管理 | 财务管理员，管理结算和支付 |

#### 权限验证代码

```javascript
// 前端权限验证
const checkPermission = (requiredRole) => {
  const userRole = getCurrentUserRole();
  const roleHierarchy = {
    'super_admin': 100,
    'water_admin': 50,
    'meeting_admin': 50,
    'finance_admin': 50
  };
  
  return roleHierarchy[userRole] >= roleHierarchy[requiredRole];
};

// 菜单项权限过滤
const filterMenuByPermission = (menuItems) => {
  return menuItems.filter(item => {
    if (!item.requiredRole) return true;
    return checkPermission(item.requiredRole);
  });
};
```

### 3.3 登录认证流程

```
用户访问管理后台
    ↓
检查Token是否存在
    ├─ 存在 → 验证Token有效性
    │   ├─ 有效 → 进入管理后台
    │   └─ 无效 → 跳转登录页
    └─ 不存在 → 跳转登录页
    
登录页
    ↓
输入用户名密码
    ↓
POST /api/admin/login
    ↓
返回Token和用户信息
    ↓
存储Token到localStorage
    ↓
跳转到管理后台首页
```

### 6.2 迁移步骤（以水站领水记录为例）

#### 步骤1：提取核心业务逻辑

```javascript
// 从原始代码中提取领水记录管理逻辑
// 原始文件：Service_WaterManage/frontend/admin.html（第1531-2500行）

const waterPickupsModule = {
  data() {
    return {
      pickups: [],
      filters: {},
      pagination: { page: 1, pageSize: 20 }
    }
  },
  
  methods: {
    async loadPickups() {
      const res = await fetch('/api/user/office-pickups');
      this.pickups = await res.json();
    },
    
    async confirmPickup(id) {
      await fetch(`/api/pickup/${id}/confirm`, { method: 'POST' });
      await this.loadPickups();
    }
  }
};
```

#### 步骤2：创建独立模块文件

```javascript
// 创建文件：portal/admin/modules/water-pickups.js

export default {
  name: 'WaterPickupsModule',
  
  template: `
    <div class="module-container">
      <!-- 筛选器 -->
      <div class="filters">...</div>
      
      <!-- 列表 -->
      <div class="list">...</div>
      
      <!-- 分页 -->
      <div class="pagination">...</div>
    </div>
  `,
  
  setup() {
    // 使用Composition API重写
    const pickups = ref([]);
    const loading = ref(false);
    
    const loadPickups = async () => {
      loading.value = true;
      try {
        const res = await fetch('/api/user/office-pickups');
        pickups.value = await res.json();
      } finally {
        loading.value = false;
      }
    };
    
    return { pickups, loading, loadPickups };
  }
};
```

#### 步骤3：集成到统一管理后台

```javascript
// 在统一管理后台主页面中集成
import WaterPickupsModule from './modules/water-pickups.js';

// 在模板中使用
<div v-if="currentModule === 'water-pickups'">
  <WaterPickupsModule />
</div>
```
