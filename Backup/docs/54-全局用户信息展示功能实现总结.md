## 一、功能概述

### 1.1 核心功能
- ✅ 所有页面右上角统一显示用户信息
- ✅ 点击用户信息显示下拉菜单
- ✅ 下拉菜单提供常用操作（返回首页、管理后台、我的余额、修改密码、退出登录）
- ✅ 跨页面自动同步用户状态
- ✅ 响应式设计，适配移动端

### 1.2 用户故事
> 作为系统用户，我希望在登录后无论进入哪个页面，都能在页面右上角看到我的个人信息，包括头像、姓名、部门等，方便我了解当前登录状态和进行相关操作。

## 二、实现方案

### 2.1 技术架构

```
用户登录 → 获取用户信息 → 存储到localStorage → Header组件读取 → 显示用户信息
                                                           ↓
                                                        跨页面自动同步
```

## 三、核心代码

### 3.1 UserHeader.js

#### 核心功能
1. **用户信息加载**
   - 从localStorage读取用户信息
   - 解析并显示用户信息
   - 监听storage事件实现跨页面同步

2. **用户菜单**
   - 点击用户信息显示/隐藏菜单
   - 提供常用操作链接
   - 点击外部区域自动关闭菜单

3. **退出登录**
   - 显示确认对话框
   - 清除localStorage中的用户信息
   - 跳转到首页

### 3.2 UserHeader.css

#### 核心样式
1. **Header样式**
   - 渐变背景色
   - 固定在页面顶部
   - 响应式布局

2. **用户信息样式**
   - 用户头像、姓名、部门
   - 管理员徽章
   - 下拉菜单

3. **响应式样式**
   - 移动端简化显示
   - 平板适配

## 四、集成方式

### 4.1 在页面中引入组件

```html
<!-- 引入组件 -->
<script src="/water/components/UserHeader.js"></script>
<link rel="stylesheet" href="/water/components/UserHeader.css">
```

### 4.2 在Vue应用中注册组件

```javascript
const { createApp } = Vue;
const UserHeader = window.UserHeader;

const app = createApp({
    components: {
        'user-header': UserHeader
    }
});
```

### 4.3 在模板中使用组件

```html
<user-header></user-header>
```

    <div id="app">
        <!-- 使用组件 -->
        <user-header></user-header>

            <!-- 您的页面内容 -->
        </div>
    </div>
    
    <script>
        // Vue应用
        const { createApp } = Vue;
        const app = createApp({
            components: {
                'user-header': window.UserHeader
            }

</html>
