### 1.2 index.html - 用户领水主页面（1745行）

#### 功能模块详细清单

| 功能模块 | 具体功能 | UI组件 | API调用 | 业务规则 |
|---------|---------|--------|---------|---------|
| **办公室选择** | 选择领水办公室 | 卡片式选择器 | `GET /api/offices?is_active=1` | 必选办公室才能领水 |
| **产品展示** | 展示水产品列表 | 列表卡片 | `GET /api/products?is_active=1` | 按办公室筛选产品 |
| **数量选择** | 选择领取数量 | 数量选择器（+/-按钮） | 无 | 最小1桶，最大受库存限制 |
| **领取下单** | 提交领水订单 | 提交按钮 | `POST /api/pickup` | 检查库存、生成交易记录 |
| **领水记录** | 查看历史领水记录 | 列表展示 | `GET /api/user/office-pickups` | 显示最近100条记录 |
| **支付二维码** | 展示支付二维码 | 图片展示 | `GET /api/config/payment-qr` | 从后端获取支付二维码 |
| **返回首页** | 返回Portal首页 | Home按钮 | 无 | 跳转到portal/index.html |

#### 核心业务逻辑

```javascript
// 主要Vue方法
methods: {
    async loadData()           // 加载办公室和产品数据
    async loadPaymentQr()      // 加载支付二维码
    async loadRecordList()     // 加载领水记录
    async submitOrder()        // 提交领水订单
    async confirmPickup()      // 确认领水
    goHome()                   // 返回首页
}
```

#### 数据流向

```
用户选择办公室 → 加载该办公室的产品列表 → 用户选择产品和数量 → 
提交订单（POST /api/pickup）→ 生成交易记录 → 显示领水成功 → 
更新领水记录列表
```

```vue
<template>
  <div class="order-list">
    <!-- Tab切换 -->
    <div class="tab-container">
      <button v-for="tab in tabs" 
              :key="tab.value"
              @click="activeTab = tab.value"
              :class="activeTab === tab.value ? 'active' : ''">
        {{ tab.label }}
      </button>
    </div>

    <!-- 筛选器 -->
    <div class="filter-container">
      <input v-model="keyword" placeholder="搜索..." />
      <select v-model="statusFilter">
        <option value="">全部状态</option>
        <option v-for="status in statusOptions" 
                :key="status.value" 
                :value="status.value">
          {{ status.label }}
        </option>
      </select>
      <input type="date" v-model="dateFilter" />
    </div>

    <!-- 订单列表 -->
    <div class="list-container">
      <div v-for="order in filteredOrders" 
           :key="order.id" 
           class="order-card"
           @click="viewDetail(order)">
        <div class="order-header">
          <span class="order-no">{{ order.order_no }}</span>
          <span :class="`status-${order.status}`">{{ order.status_text }}</span>
        </div>
        <div class="order-body">
          <div>{{ order.product_name || order.room_name }}</div>
          <div>{{ order.created_at }}</div>
          <div class="amount">¥{{ order.amount }}</div>
        </div>
      </div>
    </div>

    <!-- 详情模态框 -->
    <OrderDetailModal v-if="selectedOrder" 
                      :order="selectedOrder" 
                      @close="selectedOrder = null" />
  </div>
</template>

<script>
export default {
  props: {
    type: String,  // 'water' | 'meeting'
    tabs: Array,   // Tab配置
    statusOptions: Array,  // 状态选项
    apiUrl: String  // API地址
  },
  data() {
    return {
      activeTab: this.tabs[0]?.value,
      keyword: '',
      statusFilter: '',
      dateFilter: '',
      orders: [],
      selectedOrder: null
    }
  },
  computed: {
    filteredOrders() {
      let result = this.orders;
      if (this.keyword) {
        result = result.filter(o => 
          o.order_no.includes(this.keyword) || 
          (o.product_name || o.room_name).includes(this.keyword)
        );
      }
      if (this.statusFilter) {
        result = result.filter(o => o.status === this.statusFilter);
      }
      if (this.dateFilter) {
        result = result.filter(o => o.created_at.startsWith(this.dateFilter));
      }
      return result;
    }
  },
  async mounted() {
    const res = await fetch(this.apiUrl);
    this.orders = await res.json();
  }
}
</script>
```

- 水站领水记录查询

```javascript
// 前端获取业务规则
async loadBusinessRules() {
    const res = await fetch('/api/business-rules');
    this.rules = await res.json();
}

// 使用业务规则
calculateFee() {
    if (this.userType === 'internal') {
        return this.duration * this.room.price * this.rules.internal_discount_rate;
    } else {
        return this.duration * this.room.price;
    }
}
```

- 支持不同办公室不同规则
