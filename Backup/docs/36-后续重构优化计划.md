# utils/database.py

from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker

class DatabaseManager:
    _instances = {}
    
    @classmethod
    def get_instance(cls, db_name='waterms'):
        if db_name not in cls._instances:
            engine = create_engine(f'sqlite:///./{db_name}.db')
            cls._instances[db_name] = sessionmaker(bind=engine)
        return cls._instances[db_name]

## 十三、关键决策记录

### 决策1: 采用"核心域+服务域"分层架构

**背景**：
- 系统包含多个业务域
- 部分业务域跨服务共享（用户、办公室、结算）
- 部分业务域独立（水站、会议室、用餐）

**决策**：
- 核心域：统一实现，全局共享
- 服务域：独立实现，依赖核心域

**理由**：
- 避免重复实现
- 统一业务逻辑
- 便于维护和扩展

**影响**：
- 需要建立清晰模块边界
- 需要定义模块间依赖规范
- 需要建立服务层

### 决策2: 不继续迁移Package模型

**背景**：
- Package是水站特有业务
- 当前应优先完成核心域模块化

**决策**：
- 暂不迁移Package模型
- 优先完成办公室管理、结算管理模块

**理由**：
- 核心域优先原则
- 降低风险
- 逐步演进

**影响**：
- Package相关代码暂时保留在 models_unified.py
- 后续在 services/water/ 中统一处理
