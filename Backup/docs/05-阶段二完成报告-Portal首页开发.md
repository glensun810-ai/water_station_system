## 三、技术实现

### 3.1 技术栈
- **前端框架**: Vue 3（Composition API）
- **样式系统**: 自定义CSS + 设计系统样式
- **响应式布局**: Flex + Grid
- **移动端适配**: 媒体查询 + 安全区域

### 3.2 核心功能

#### 响应式布局
- **PC端**: 三列网格布局
- **移动端**: 单列布局 + 底部固定导航
- **自适应**: 根据屏幕宽度自动调整

### 3.3 API接口

| 接口 | 用途 | 返回数据 |
|------|------|---------|
| `GET /api/unified/user/{user_id}/balance` | 获取水站余额 | balance_prepaid |
| `GET /api/unified/settlement/summary` | 获取待结算金额 | total_unsettled |
| `GET /api/meeting/user/{user_id}/free-hours` | 获取免费时长 | free_hours |

## 八、使用指南

### 8.1 访问Portal首页

**URL**: `http://localhost:8000/`

### 8.2 主要功能

1. **查看个人信息**: 页面加载后自动显示余额、待结算、免费时长等信息
2. **使用水站服务**: 点击"水站服务"卡片或底部的"水站"导航
3. **预约会议室**: 点击"会议室服务"卡片或底部的"会议室"导航
4. **快速操作**: 在快捷操作区域点击"前往领水"或"前往预约"

### 8.3 移动端操作

1. **底部导航**: 点击底部图标切换不同服务
2. **用户菜单**: 点击"我的"打开侧边菜单
3. **退出登录**: 在侧边菜单中点击"退出登录"
