## 一、需求分析

### 1.1 用户故事
> 作为系统用户，我希望在登录后无论进入哪个页面，都能在页面右上角看到我的个人信息，包括头像、姓名、部门等，方便我了解当前登录状态和进行相关操作。

### 1.2 核心需求
1. **统一登录体验**: 用户在首页登录后，全局保持登录状态
2. **全局用户信息展示**: 所有页面右上角统一显示用户信息
3. **一致性体验**: 所有页面使用相同的用户信息展示方式
4. **便捷操作**: 用户可以随时退出登录、访问个人设置等

### 1.3 用户旅程
```
访问系统 → 首页(未登录) → 登录 → 首页(已登录) → 进入任意子页面 → 右上角显示用户信息
                                                            ↓
                                                    可随时退出/查看个人信息
```

## 二、产品方案设计

### 2.1 方案选择

#### 推荐方案: 全局Header组件

**设计理念**:
- 所有页面共享统一的Header组件
- Header组件负责用户信息展示和全局导航
- 使用localStorage存储用户状态，实现跨页面同步

**优点**:
- ✅ 统一的用户体验
- ✅ 易于维护和更新
- ✅ 响应式设计友好
- ✅ 代码复用率高
- ✅ 用户状态管理集中

**缺点**:
- ⚠️ 需要在所有页面引入组件
- ⚠️ 可能需要调整部分页面布局

<script>
export default {
  name: 'UserHeader',
  data() {
    return {
      userInfo: null,
      showUserMenu: false
    };
  },
  mounted() {
    this.loadUserInfo();
    // 监听localStorage变化
    window.addEventListener('storage', this.handleStorageChange);
    document.addEventListener('click', this.handleClickOutside);
  },
  beforeUnmount() {
    window.removeEventListener('storage', this.handleStorageChange);
    document.removeEventListener('click', this.handleClickOutside);
  },
  methods: {
    loadUserInfo() {
      const userInfo = localStorage.getItem('userInfo');
      if (userInfo) {
        this.userInfo = JSON.parse(userInfo);
      }
    },
    handleStorageChange(event) {
      if (event.key === 'userInfo') {
        this.loadUserInfo();
      }
    },
    toggleUserMenu() {
      this.showUserMenu = !this.showUserMenu;
    },
    handleClickOutside(event) {
      if (!event.target.closest('.user-header')) {
        this.showUserMenu = false;
      }
    },
    handleLogout() {
      if (confirm('确定要退出登录吗？')) {
        localStorage.removeItem('token');
        localStorage.removeItem('userInfo');
        window.location.href = '/portal/index.html';
      }
    }
  }
};
</script>

### 7.2 数据缓存

```javascript
// 缓存用户信息，减少API调用
const loadUserInfo = async () => {
  const cached = localStorage.getItem('userInfo');
  if (cached) {
    userInfo.value = JSON.parse(cached);
  }
  
  // 后台刷新用户信息
  const fresh = await fetch('/api/v2/users/me');
  if (fresh.ok) {
    const data = await fresh.json();
    userInfo.value = data;
    localStorage.setItem('userInfo', JSON.stringify(data));
  }
};
```
