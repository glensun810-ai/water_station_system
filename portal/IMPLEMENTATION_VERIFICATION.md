## 二、面包屑导航结构验证

### 标准结构

所有管理后台页面统一采用三层级面包屑：

```
🏢 AI产业集群空间服务（固定根节点）
   ↓
⚙️ 管理中心（第2层级，链接到首页）
   ↓
[功能图标] [功能名称]（第3层级，当前页面）
```

### 实际配置示例

**办公室管理页面：**
```html
<global-header :breadcrumbs="[
  {text: '管理中心', icon: '⚙️', url: '/portal/index.html'},
  {text: '办公室管理', icon: '🏢', url: '/portal/admin/offices.html'}
]">
</global-header>
```

**用户管理页面：**
```html
<global-header :breadcrumbs="[
  {text: '管理中心', icon: '⚙️', url: '/portal/index.html'},
  {text: '用户管理', icon: '👥', url: '/portal/admin/users.html'}
]">
</global-header>
```
