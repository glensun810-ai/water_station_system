## 🎨 设计规范

### 图标使用规范

| 功能 | 图标 | 用途 |
|------|------|------|
| 首页 | 🏠 | 全局返回首页 |
| 预约 | 📋 | 预约页面、预约功能 |
| 日历 | 📅 | 日历视图 |
| 我的预约 | 📝 | 我的预约页面 |
| 管理 | ⚙️ | 管理后台入口 |
| 新建 | + | 新建预约按钮 |

### 颜色使用规范

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主按钮 | 蓝色 (bg-blue-600) | 主要操作 |
| 行动按钮 | 绿色 (bg-green-500) | 创建、添加 |
| 当前页标识 | 蓝色背景白字 | 标识当前位置 |
| 普通按钮 | 白色/灰色边框 | 次要操作 |
| 推荐标签 | 绿色文字 | 功能推荐 |

### 布局规范

```
导航栏结构：
┌────────────────────────────────────┐
│ [首页] | [标题]      [功能1][功能2]│
└────────────────────────────────────┘

间距规范：
- 左右区域间距：space-x-3 或 space-x-4
- 按钮内边距：px-4 py-2
- 分隔线：h-6 w-px bg-slate-300

层级规范：
- 导航栏 z-index: 50
- 下拉菜单 z-index: 40
- 弹窗 z-index: 50
```
