#### 📌 API 1: 优惠详情计算接口

**路径**: `POST /api/unified/promotion/calculate`

**功能**：计算优惠详情，返回详细的优惠计算过程

**请求参数**:
```json
{
  "product_id": 1,
  "quantity": 15
}
```

**响应示例**:
```json
{
  "product_id": 1,
  "product_name": "桶装水 18L",
  "unit_price": 15.0,
  "quantity": 15,
  "gift_qty": 1,
  "paid_qty": 14,
  "total_amount": 210.0,
  "original_amount": 225.0,
  "savings": 15.0,
  "discount_rate": 6.67,
  "message": "已优惠 1 件",
  "next_gift_progress": {
    "current": 5,
    "need": 5,
    "percent": 50
  }
}
```

**实现文件**: `backend/api_unified.py` (Line 898-965)

**核心逻辑**:
- 买 N 赠 M 计算：`cycles = quantity // trigger_qty; gift_qty = cycles * gift_qty`
- 距离下次优惠进度计算
- 优惠率和节省金额计算

// 计算赠送数量
totalGiftQty() {
    return this.products.reduce((sum, p) => {
        const qty = this.cart[p.id] || 0;
        if (qty > 0 && p.promo_threshold && p.promo_gift) {
            sum += Math.floor(qty / p.promo_threshold) * p.promo_gift;
        }
        return sum;
    }, 0);
}
```

const toggleProductPromo = async (product, enable) => {
    try {
        const res = await fetch(`${API_BASE}/products/${product.id}/promo-toggle?enable=${enable}`, {
            method: 'PUT',
            headers: {
                'Authorization': `Bearer ${localStorage.getItem('token')}`
            }
        });
        
        if (res.ok) {
            const result = await res.json();
            product.promo_threshold = result.promo_threshold;
            product.promo_gift = result.promo_gift;
            showMessage(`${result.product_name} ${result.message}`, 'success');
        }
    } catch (error) {
        showMessage('网络错误，请重试', 'error');
    }
};
```

## 三、技术实现亮点

### 3.1 零侵入式设计

- ✅ **无数据库迁移**: 复用现有 `promo_threshold` 和 `promo_gift` 字段
- ✅ **无核心逻辑改动**: 复用 `discount_strategy.py` 中的买赠计算逻辑
- ✅ **向后兼容**: 所有新功能均为增强，不影响现有业务流程

### 3.2 前后端分离架构

- **后端职责**: 提供清晰的 API 接口，负责数据计算和业务逻辑
- **前端职责**: 负责展示优化和用户交互，90% 的改动在前端展示层

### 3.3 性能优化

- **前端缓存**: 计算属性基于 Vue 响应式系统，自动缓存
- **防抖处理**: 数量选择器输入变化时，避免频繁计算
- **API 设计**: 统计接口支持时间范围筛选，避免全量查询

### 3.4 用户体验设计

- **渐进式展示**: 
  - 第一层：产品列表展示优惠标签
  - 第二层：选择数量后显示智能提醒
  - 第三层：结算时展示完整优惠明细
- **色彩心理学**:
  - 橙色系（orange-500/600）：温暖、亲和，突出优惠氛围
  - 绿色系（green-600）：环保、省钱，强调节省金额
  - 红色系（red-600）：醒目、重要，高亮赠送数量
