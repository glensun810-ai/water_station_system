## 一、产品定位与核心价值

### 1\. 产品定位

本会议室服务模块是**AI 产业集群空间服务系统**的核心功能子模块，面向园区企业员工、入驻团队、商务访客，提供**线上自助预约、智能时段管理、订单全生命周期管理、场地资源管控**的标准化会议室服务，替代线下人工登记，实现园区会议室资源数字化、高效化、无人化管理。

### 2\. 核心价值

1. **提升资源利用率**：可视化展示会议室状态、时段、配置，避免资源冲突与闲置

2. **降低管理成本**：自动预约、自动核验、自动统计，减少人工运维

3. **优化用户体验**：全流程线上操作，支持快速预约、取消、续费、查看订单

4. **适配产业园区**：支持企业认证、权限管控、收费 / 免费双模式

### 3\. 目标用户

|用户类型|核心权限|核心操作|
|---|---|---|
|普通用户（企业员工）|预约、查看、取消、支付、查看订单|自主预约会议室、按时段付费、查看使用规则|
|管理员（园区运营）|全部管理权限|会议室增删改查、订单审核、数据统计、违规管理|

## 三、页面功能详细设计（HTML5 网页版）

### 1\. 会议室列表页（核心入口页）

#### 页面功能

1. **顶部筛选栏**

    - 日期选择：今日 / 明日 / 自定义日期（日历选择器）

    - 容量筛选：1\-5 人、6\-10 人、11\-20 人、20 人以上

    - 设备筛选：投影仪、视频会议、白板、WIFI、空调

    - 排序：默认排序 / 价格从低到高 / 距离最近

2. **会议室卡片列表**
每一张卡片展示：

    - 会议室封面图

    - 会议室名称（如：A101 智能会议室）

    - 容纳人数

    - 基础配置（图标形式：投影、视频会议等）

    - 当前状态：可预约 / 已满 / 维护中

    - 参考价格（元 / 30 分钟 / 元 / 小时）

    - 快速预约按钮

3. **顶部快捷入口**

    - 我的订单

    - 取消订单

    - 使用规则

    - 客服 / 帮助中心

#### HTML5 交互要求

- 响应式布局，手机 / PC 自适应

- 卡片点击进入详情页

- 筛选条件实时刷新列表

---

### 2\. 会议室详情页（核心业务页）

#### 页面模块

1. **基础信息区**

    - 轮播图 / 实景图

    - 会议室名称、位置、楼层、面积

    - 容纳人数、开放时间

    - 价格展示（小时价 / 半小时价）

    - 状态：可预约 / 不可预约

2. **配置与服务**

    - 硬件设备：高清投影、4K 显示屏、视频会议系统、白板、音响、麦克风

    - 基础服务：免费 WIFI、空调、饮水、电源插座

    - 增值服务（可选）：打印、茶水、技术支持、清洁服务

3. **使用规则（必展示）**

    - 预约提前时间（如：至少提前 30 分钟预约）

    - 取消规则（如：使用前 2 小时可免费取消，超时扣 30% 费用）

    - 迟到规则（迟到 15 分钟自动取消）

    - 卫生要求、设备使用规范

    - 违规处罚规则

4. **可预约时段选择器（核心）**

    - 日历：选择使用日期

    - 时段轴：24 小时制，以 30 分钟为最小单位

    - 颜色区分：

        - 绿色：可预约

        - 灰色：已被预约

        - 黄色：我的预约

    - 支持**连续多选时段**（如：9:00\-11:30）

    - 实时计算：时长、费用

5. **操作按钮**

    - 立即预约

    - 联系管理员

    - 收藏会议室

#### 业务规则

- 最小预约单位：30 分钟

- 最大预约时长：系统可配置（默认 4 小时 / 次）

- 同一用户单日预约上限：可配置

- 开放时段外不可预约

## 四、核心业务流程

### 1\. 用户预约流程

```Plain Text
登录系统 → 进入会议室服务 → 选择日期/筛选 → 选择会议室 → 选择时段 → 确认信息 → 提交订单 → 支付 → 生成待使用订单 → 按时使用 → 自动完成 → 可评价
```

### 2\. 取消订单流程

```Plain Text
我的订单 → 待使用 → 取消订单 → 系统校验时间 → 计算退款 → 确认取消 → 退款到账 → 状态变为已取消
```

### 3\. 管理员审核流程（可选）

```Plain Text
用户提交订单 → 管理员后台审核 → 通过/拒绝 → 用户收到通知 → 支付/重新预约
```
