## 一、服务器环境准备

### 1.1 系统初始化（推荐全新 Ubuntu 22.04）

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装基础工具
sudo apt install -y curl wget git vim ufw software-properties-common

# 安装 Python 3.11
sudo add-apt-repository ppa:deadsnakes/ppa -y
sudo apt install -y python3.11 python3.11-venv python3.11-dev python3.11-distutils

# 设置默认 Python
sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.11 1
curl -sS https://bootstrap.pypa.io/get-pip.py | python3.11

# 创建项目用户（可选）
sudo useradd -m -s /bin/bash admin
sudo usermod -aG sudo admin
```

### 1.2 安装 Nginx

```bash
# 安装 Nginx
sudo apt install -y nginx

# 启动并设置开机自启
sudo systemctl start nginx
sudo systemctl enable nginx

# 检查状态
sudo systemctl status nginx

# 开放端口
sudo ufw allow 22    # SSH
sudo ufw allow 80    # HTTP
sudo ufw allow 443   # HTTPS
sudo ufw --force enable
```

### 1.3 安装 Certbot (Let's Encrypt)

```bash
# 安装 Certbot
sudo snap install --classic certbot

# 创建软链接
sudo ln -sf /snap/bin/certbot /usr/bin/certbot

# 测试 Certbot
certbot --version
```

## 五、后端服务配置

### 5.1 创建系统服务（systemd）

**文件**: `/etc/systemd/system/water-api.service`

```ini
[Unit]
Description=Water Management API Service
After=network.target

[Service]
User=www-data
Group=www-data
WorkingDirectory=/var/www/jhw-ai.com/Service_WaterManage/backend
Environment="PATH=/var/www/jhw-ai.com/Service_WaterManage/backend/venv/bin"
Environment="PYTHONPATH=/var/www/jhw-ai.com/Service_WaterManage/backend"
ExecStart=/var/www/jhw-ai.com/Service_WaterManage/backend/venv/bin/python -m uvicorn main:app --host 127.0.0.1 --port 8000
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

### 5.2 启动服务

```bash
# 重新加载 systemd
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start water-api

# 设置开机自启
sudo systemctl enable water-api

# 查看状态
sudo systemctl status water-api

# 查看日志
sudo journalctl -u water-api -f
```

## 六、Nginx 配置

### 6.1 创建 Nginx 配置文件

**文件**: `/etc/nginx/sites-available/jhw-ai.com-water`

```nginx
server {
    listen 80;
    server_name jhw-ai.com;

    # 重定向到 HTTPS
    return 301 https://$host$request_uri;
}

server {
    listen 443 ssl http2;
    server_name jhw-ai.com;

    # SSL 证书配置（Certbot 自动添加）
    ssl_certificate /etc/letsencrypt/live/jhw-ai.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/jhw-ai.com/privkey.pem;
    ssl_session_timeout 1d;
    ssl_session_cache shared:SSL:50m;
    ssl_session_tickets off;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256:ECDHE-ECDSA-AES256-GCM-SHA384:ECDHE-RSA-AES256-GCM-SHA384;
    ssl_prefer_server_ciphers off;

    # 用户端 - /water/
    location /water/ {
        alias /var/www/jhw-ai.com/water/;
        index index.html;
        try_files $uri $uri/ /water/index.html;
        
        # 缓存静态资源
        location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
            expires 30d;
            add_header Cache-Control "public, immutable";
        }
    }

    # 管理后台 - /water-admin/
    location /water-admin/ {
        alias /var/www/jhw-ai.com/water-admin/;
        index admin.html;
        try_files $uri $uri/ /water-admin/admin.html;
        
        # 缓存静态资源
        location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
            expires 30d;
            add_header Cache-Control "public, immutable";
        }
    }

    # API 代理 - /api/
    location /api/ {
        proxy_pass http://127.0.0.1:8000/api/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        
        # 超时设置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }

    # 健康检查
    location /health {
        proxy_pass http://127.0.0.1:8000/api/health;
        proxy_set_header Host $host;
    }
}
```

### 6.2 启用配置

```bash
# 创建软链接
sudo ln -s /etc/nginx/sites-available/jhw-ai.com-water /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重载 Nginx
sudo systemctl reload nginx
```

## 七、SSL 证书配置

### 7.1 申请 Let's Encrypt 证书

```bash
# 确认域名已解析到服务器
ping jhw-ai.com

# 申请证书（需要 Nginx 已配置并运行）
sudo certbot --nginx -d jhw-ai.com -d www.jhw-ai.com

# 输入邮箱并同意服务条款
# 选择是否重定向 HTTP 到 HTTPS（建议选择 2 重定向）
```

### 7.2 证书自动续期

```bash
# 测试续期
sudo certbot renew --dry-run

# 添加 cron 任务（Certbot 通常自动配置）
sudo crontab -e
# 添加以下行（每天凌晨检查续期）
0 0 * * * certbot renew --quiet --deploy-hook "systemctl reload nginx"
```
