## 六、问题说明

### 6.1 前端弹窗无法关闭问题

**原因**: 后端 `/api/admin/inventory/*` 系列 API 需要 super_admin 权限，当前用户为 admin 角色，导致 API 调用失败（403），前端页面渲染阻塞。

**临时解决方案**: 禁用相关弹窗和按钮，确保后台管理系统可以正常启动。

**后续修复方向**:
1. 修改后端 API 权限配置，允许 admin 角色访问库存管理 API
2. 或者在前端添加更健壮的错误处理，确保 API 失败不影响页面渲染

### 6.2 预警配置弹窗问题

**原因**: Vue 模板中 `v-if/v-else` 链不完整，导致模板编译失败。

**临时解决方案**: 注释掉预警配置弹窗代码。
