## 一、问题背景

### 1.1 当前状态

**已完成部分**:
- ✅ 数据库表结构完整（meeting_rooms, meeting_bookings）
- ✅ 后端API完整实现（10个端点）
- ✅ 管理后台功能完整
- ✅ 示例数据已创建

**存在问题**:
- ❌ 用户端使用硬编码数据，无法显示真实会议室
- ❌ 用户端API端点错误，预约功能不可用
- ❌ 缺少内部/外部用户区分机制
- ❌ 缺少会员价格优惠机制
- ❌ 缺少办公室关联功能

### 1.2 业务需求

**核心需求**:
1. **内部用户**（AI产业集群会员办公室）
   - 从办公室列表选择
   - 享受会员优惠价格
   - 关联办公室账户

2. **外部用户**（非会员）
   - 填写联系信息
   - 按标准价格计费
   - 无办公室关联

**关键要求**:
- 办公室数据统一使用水站管理中的办公室模块
- 数据库设计要兼容现有系统
- 遵循零风险开发原则

# ==================== 办公室管理 API ====================

@router.get("/offices")
def get_offices():
    """
    获取办公室列表（供内部用户选择）
    从水站管理系统的offices表获取数据
    """
    try:
        from sqlalchemy import create_engine, text
        from sqlalchemy.orm import sessionmaker
        
        # 连接到水站管理数据库
        db_path = os.path.join(
            os.path.dirname(__file__), 
            '../../Service_WaterManage/waterms.db'
        )
        db_path = os.path.abspath(db_path)
        
        engine = create_engine(f"sqlite:///{db_path}")
        SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)
        db = SessionLocal()
        
        try:
            # 查询活跃的办公室列表
            query = text("""
                SELECT id, name, room_number, leader_name 
                FROM offices 
                WHERE is_active = 1 
                ORDER BY name
            """)
            
            result = db.execute(query)
            offices = []
            
            for row in result:
                offices.append({
                    "id": row.id,
                    "name": row.name,
                    "room_number": row.room_number,
                    "leader_name": row.leader_name,
                    "display_name": f"{row.name} ({row.room_number})"
                })
            
            return offices
        finally:
            db.close()
    except Exception as e:
        raise HTTPException(
            status_code=500, 
            detail=f"获取办公室列表失败: {str(e)}"
        )

#### 修改4: 修改创建预约逻辑

**位置**: `api_meeting.py:318-418` 的 `create_booking` 函数

**关键修改点**:

```python
@router.post("/bookings", response_model=MeetingBookingResponse)
def create_booking(booking: MeetingBookingCreate, db: Session = Depends(get_db)):
    """创建预约"""
    try:
        from sqlalchemy import text
        
        # ✅ 1. 验证用户类型
        if booking.user_type not in ['internal', 'external']:
            raise HTTPException(status_code=400, detail="用户类型必须是 internal 或 external")
        
        # ✅ 2. 验证内部用户必须提供office_id
        if booking.user_type == 'internal':
            if not booking.office_id:
                raise HTTPException(status_code=400, detail="内部用户必须选择办公室")
            
            # 验证办公室是否存在（可选）
            # 这里可以调用办公室验证逻辑
        
        # ✅ 3. 获取会议室信息
        room = db.execute(text(f"SELECT * FROM meeting_rooms WHERE id = {booking.room_id}")).fetchone()
        if not room:
            raise HTTPException(status_code=404, detail="会议室不存在")
        
        # ✅ 4. 检查时间冲突
        conflict_query = text("""
            SELECT id FROM meeting_bookings 
            WHERE room_id = :room_id 
            AND booking_date = :booking_date
            AND status IN ('pending', 'confirmed')
            AND (
                (start_time <= :start_time AND end_time > :start_time)
                OR (start_time < :end_time AND end_time >= :end_time)
                OR (start_time >= :start_time AND end_time <= :end_time)
            )
        """)
        conflicts = db.execute(conflict_query, {
            "room_id": booking.room_id,
            "booking_date": str(booking.booking_date),
            "start_time": booking.start_time,
            "end_time": booking.end_time
        }).fetchone()
        
        if conflicts:
            raise HTTPException(status_code=400, detail="该时间段已被预约")
        
        # ✅ 5. 计算时长
        start_hour = int(booking.start_time.split(':')[0])
        end_hour = int(booking.end_time.split(':')[0])
        duration = end_hour - start_hour
        
        # ✅ 6. 根据用户类型计算费用
        if booking.user_type == 'internal':
            # 内部用户使用会员价
            price = float(room.member_price_per_hour) if room.member_price_per_hour else float(room.price_per_hour) * 0.8
        else:
            # 外部用户使用标准价
            price = float(room.price_per_hour)
        
        total_fee = duration * price
        
        # ✅ 7. 创建预约
        booking_no = generate_booking_no()
        now = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
        
        query = text("""
            INSERT INTO meeting_bookings 
            (booking_no, room_id, room_name, user_type, office_id, user_name, user_phone, department, 
             booking_date, start_time, end_time, duration, meeting_title, 
             attendees_count, total_fee, status, created_at, updated_at)
            VALUES 
            (:booking_no, :room_id, :room_name, :user_type, :office_id, :user_name, :user_phone, :department,
             :booking_date, :start_time, :end_time, :duration, :meeting_title,
             :attendees_count, :total_fee, :status, :created_at, :updated_at)
        """)
        
        result = db.execute(query, {
            "booking_no": booking_no,
            "room_id": booking.room_id,
            "room_name": room.name,
            "user_type": booking.user_type,
            "office_id": booking.office_id,
            "user_name": booking.user_name,
            "user_phone": booking.user_phone,
            "department": booking.department,
            "booking_date": str(booking.booking_date),
            "start_time": booking.start_time,
            "end_time": booking.end_time,
            "duration": duration,
            "meeting_title": booking.meeting_title,
            "attendees_count": booking.attendees_count,
            "total_fee": total_fee,
            "status": "pending",
            "created_at": now,
            "updated_at": now
        })
        db.commit()
        
        booking_id = result.lastrowid
        
        return {
            "id": booking_id,
            "booking_no": booking_no,
            "room_id": booking.room_id,
            "room_name": room.name,
            "user_type": booking.user_type,
            "office_id": booking.office_id,
            "user_name": booking.user_name,
            "user_phone": booking.user_phone,
            "department": booking.department,
            "booking_date": booking.booking_date,
            "start_time": booking.start_time,
            "end_time": booking.end_time,
            "duration": duration,
            "meeting_title": booking.meeting_title,
            "attendees_count": booking.attendees_count,
            "total_fee": total_fee,
            "status": "pending",
            "created_at": datetime.now()
        }
    except HTTPException:
        raise
    except Exception as e:
        db.rollback()
        raise HTTPException(status_code=500, detail=f"创建预约失败: {str(e)}")
```

### 5.2 JavaScript逻辑

**核心功能实现**:

```javascript
const { createApp, ref, computed, onMounted, watch } = Vue;

createApp({
    setup() {
        const API_BASE = 'http://localhost:8000/api/meeting';
        
        // ==================== 状态变量 ====================
        const userType = ref('external');  // 用户类型
        const offices = ref([]);  // 办公室列表
        const selectedOfficeId = ref(null);  // 选中的办公室
        
        const rooms = ref([]);  // 会议室列表
        const selectedRoom = ref(null);  // 选中的会议室
        const loading = ref(false);  // 加载状态
        const error = ref(null);  // 错误状态
        
        const contactName = ref('');  // 外部用户姓名
        const contactPhone = ref('');  // 外部用户电话
        
        const booking = ref({
            date: '',
            timeSlot: '',
            topic: '',
            attendees: 0
        });
        
        // ==================== 数据加载 ====================
        
        // 加载办公室列表
        const loadOffices = async () => {
            try {
                const res = await fetch(`${API_BASE}/offices`);
                if (res.ok) {
                    offices.value = await res.json();
                }
            } catch (e) {
                console.error('加载办公室失败:', e);
            }
        };
        
        // 加载会议室列表
        const loadRooms = async () => {
            loading.value = true;
            error.value = null;
            
            try {
                const res = await fetch(`${API_BASE}/rooms?is_active=true`);
                if (res.ok) {
                    const data = await res.json();
                    rooms.value = data.map(room => ({
                        id: room.id,
                        name: room.name,
                        location: room.location,
                        capacity: room.capacity,
                        facilities: JSON.parse(room.facilities || '[]'),
                        price_per_hour: room.price_per_hour,
                        member_price_per_hour: room.member_price_per_hour,
                        free_hours_per_month: room.free_hours_per_month
                    }));
                } else {
                    error.value = '加载会议室列表失败';
                }
            } catch (e) {
                error.value = '网络错误，请稍后重试';
                console.error('加载失败:', e);
            } finally {
                loading.value = false;
            }
        };
        
        // ==================== 用户类型选择 ====================
        
        const selectUserType = (type) => {
            userType.value = type;
            selectedRoom.value = null;
            resetBooking();
            
            if (type === 'internal') {
                loadOffices();
            }
        };
        
        // ==================== 价格计算 ====================
        
        const calculatePrice = () => {
            if (!selectedRoom.value || !booking.value.timeSlot) return 0;
            
            const slots = {
                '09:00-12:00': 3,
                '14:00-18:00': 4,
                '19:00-21:00': 2
            };
            
            const hours = slots[booking.value.timeSlot] || 0;
            
            // 根据用户类型选择价格
            const price = userType.value === 'internal' 
                ? selectedRoom.value.member_price_per_hour 
                : selectedRoom.value.price_per_hour;
            
            return hours * price;
        };
        
        // ==================== 提交预约 ====================
        
        const canSubmit = computed(() => {
            // 基础验证
            if (!selectedRoom.value || 
                !booking.value.date || 
                !booking.value.timeSlot || 
                booking.value.attendees <= 0) {
                return false;
            }
            
            // 内部用户必须选择办公室
            if (userType.value === 'internal' && !selectedOfficeId.value) {
                return false;
            }
            
            // 外部用户必须填写联系信息
            if (userType.value === 'external' && 
                (!contactName.value || !contactPhone.value)) {
                return false;
            }
            
            return true;
        });
        
        const submitBooking = async () => {
            if (!canSubmit.value) return;
            
            // 解析时间段
            const [startTime, endTime] = booking.value.timeSlot.split('-');
            
            // 构建请求数据
            const requestData = {
                room_id: selectedRoom.value.id,
                user_type: userType.value,
                office_id: userType.value === 'internal' ? selectedOfficeId.value : null,
                user_name: userType.value === 'internal' 
                    ? offices.value.find(o => o.id === selectedOfficeId.value)?.leader_name 
                    : contactName.value,
                user_phone: userType.value === 'internal' 
                    ? '' // 可以从办公室获取
                    : contactPhone.value,
                department: '', // 可选
                booking_date: booking.value.date,
                start_time: startTime,
                end_time: endTime,
                meeting_title: booking.value.topic,
                attendees_count: booking.value.attendees
            };
            
            try {
                const res = await fetch(`${API_BASE}/bookings`, {
                    method: 'POST',
                    headers: { 'Content-Type': 'application/json' },
                    body: JSON.stringify(requestData)
                });
                
                const result = await res.json();
                
                if (res.ok) {
                    alert(`预定成功！\n\n预约编号：${result.booking_no}\n会议室：${result.room_name}\n时间：${result.booking_date} ${result.start_time}-${result.end_time}\n费用：¥${result.total_fee}`);
                    resetAll();
                } else {
                    alert('预定失败：' + (result.detail || '未知错误'));
                }
            } catch (e) {
                console.error('预定失败:', e);
                alert('预定失败，请稍后重试');
            }
        };
        
        // ==================== 辅助函数 ====================
        
        const selectRoom = (room) => {
            selectedRoom.value = room;
        };
        
        const resetBooking = () => {
            booking.value = {
                date: '',
                timeSlot: '',
                topic: '',
                attendees: 0
            };
        };
        
        const resetAll = () => {
            selectedRoom.value = null;
            selectedOfficeId.value = null;
            contactName.value = '';
            contactPhone.value = '';
            resetBooking();
        };
        
        // ==================== 生命周期 ====================
        
        onMounted(() => {
            loadRooms();
        });
        
        // ==================== 返回 ====================
        
        return {
            userType,
            offices,
            selectedOfficeId,
            rooms,
            selectedRoom,
            loading,
            error,
            contactName,
            contactPhone,
            booking,
            selectUserType,
            selectRoom,
            calculatePrice,
            canSubmit,
            submitBooking,
            loadRooms,
            resetBooking
        };
    }
}).mount('#app');
```

## 七、关键决策点

### 7.1 数据库连接方式

**问题**: 会议室模块需要访问offices表（在waterms.db中）

**选项对比**:

| 方案 | 优点 | 缺点 | 推荐 |
|-----|------|------|------|
| 方案1: 连接两个数据库 | 数据统一，无冗余 | 需要管理两个连接 | ✅ 推荐 |
| 方案2: 复制offices表 | 简单快速 | 数据冗余，需同步 | ❌ 不推荐 |
| 方案3: 合并数据库 | 单一数据源 | 改动大，风险高 | ❌ 不推荐 |

**实施方案**:

```python
# 在 api_meeting.py 中连接两个数据库
def get_offices_db():
    """获取办公室数据库连接"""
    db_path = os.path.join(
        os.path.dirname(__file__), 
        '../../Service_WaterManage/waterms.db'
    )
    db_path = os.path.abspath(db_path)
    
    engine = create_engine(f"sqlite:///{db_path}")
    SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)
    return SessionLocal()
```

### 7.2 会员价格策略

**问题**: 如何设置会员价格？

**策略**:
1. **默认折扣**: 会员价为标准价的80%
2. **可配置**: 每个会议室可单独设置会员价
3. **灵活性**: 支持后续调整

**实施**:
- 数据库字段: `member_price_per_hour`
- 默认值: `price_per_hour * 0.8`
- 更新频率: 随时可调整

### 7.3 内部用户验证

**问题**: 如何验证用户是否为内部用户？

**方案对比**:

| 方案 | 优点 | 缺点 | 推荐 |
|-----|------|------|------|
| 通过办公室选择自动识别 | 简单可靠 | 无验证机制 | ✅ 推荐 |

| 验证码验证 | 防止滥用 | 用户体验差 | ❌ 不推荐 |

**实施方案**:
- 内部用户必须从办公室列表选择
- 办公室列表来自真实数据
- 自动关联办公室ID

## 十、风险与应对

### 10.1 技术风险

#### 风险1: offices表结构不兼容

**风险等级**: 中

**描述**: 会议室模块需要访问waterms.db中的offices表，但表结构可能不兼容

**应对措施**:
1. 先查询offices表结构
2. 确认必需字段：id, name, room_number, leader_name, is_active
3. 如果缺少字段，考虑添加或使用替代字段
4. 编写兼容性适配代码

**检查脚本**:
```bash
sqlite3 Service_WaterManage/waterms.db "PRAGMA table_info(offices)"
```

---

#### 风险2: 双数据库连接问题

**风险等级**: 中

**描述**: api_meeting.py需要同时连接meeting.db和waterms.db

**应对措施**:
1. 使用绝对路径配置数据库
2. 在代码中动态获取数据库路径
3. 测试数据库连接是否正常
4. 添加错误处理和日志

**示例代码**:
```python
import os

def get_offices_db():
    db_path = os.path.join(
        os.path.dirname(__file__), 
        '../../Service_WaterManage/waterms.db'
    )
    db_path = os.path.abspath(db_path)
    
    if not os.path.exists(db_path):
        raise FileNotFoundError(f"数据库文件不存在: {db_path}")
    
    engine = create_engine(f"sqlite:///{db_path}")
    return sessionmaker(bind=engine)()
```

---

#### 风险3: 会员价格配置错误

**风险等级**: 低

**描述**: 会议室会员价格设置错误或为0

**应对措施**:
1. 设置合理默认值（标准价的80%）
2. 在API返回时检查价格是否为0
3. 提供管理后台修改价格功能
4. 添加价格验证逻辑

**验证脚本**:
```bash
sqlite3 meeting.db "
  SELECT 
    id, name, 
    price_per_hour, 
    member_price_per_hour,
    CASE 
      WHEN member_price_per_hour = 0 THEN '❌ 未设置'
      ELSE '✅ 已设置'
    END as status
  FROM meeting_rooms
"
```

### 10.2 业务风险

#### 风险1: 内部用户冒用

**风险等级**: 低

**描述**: 外部用户可能选择内部用户享受优惠

**应对措施**:
1. 暂不实现严格的验证机制
2. 通过管理后台审核预约
3. 后续可添加验证码或登录机制
4. 记录预约日志，便于追溯

---

#### 风险2: 价格争议

**风险等级**: 低

**描述**: 用户对价格有疑问或争议

**应对措施**:
1. 清晰显示价格明细
2. 区分会员价和标准价
3. 提供价格说明
4. 支持价格调整和退费流程
