# 💧 交易记录删除功能 - 使用指南

## 🎯 功能概述

交易记录删除功能已上线，支持**安全删除**、**审计追溯**和**数据恢复**。

---

## 🔐 权限说明

### 谁能删除？
- ✅ **管理员（admin 角色）** - 可以删除交易记录
- ❌ **普通员工（staff 角色）** - 无法删除

### 删除验证
- 🔑 **管理员密码** - 默认：`admin123`
- 📝 **删除原因** - 选填，建议填写

---

## 📱 操作步骤

### 单条删除

1. **找到目标记录**
   - 打开"交易记录"标签
   - 使用筛选功能快速定位

2. **点击删除按钮**
   ```
   操作列 → 删除（红色）
   ```

3. **输入密码验证**
   ```
   管理员密码：●●●●●●●●
   删除原因：数据录入错误（选填）
   ```

4. **确认删除**
   - 点击"确认删除"按钮

### 批量删除

1. **勾选多条记录**
   - 使用表头复选框全选
   - 或手动勾选特定记录

2. **点击批量删除**
   ```
   底部操作栏 → 批量删除（红色按钮）
   ```

3. **输入密码和原因**
   - 同上

### 恢复已删除记录

1. **查看已删除记录**
   - 已删除记录显示红色背景和"已删除"标签
   - 或在 URL 中添加 `?include_deleted=true`

2. **点击恢复按钮**
   ```
   操作列 → 恢复（绿色）
   ```

3. **确认恢复**
   - 记录恢复正常状态

## ⚠️ 注意事项

### 删除前
- ✅ 确认是否为误操作
- ✅ 确认是否有备份需要
- ✅ 填写删除原因便于追溯

### 删除后
- 📊 默认列表不再显示
- 🔍 可通过 `include_deleted=true` 查看
- ♻️ 支持恢复，无时间限制

### 审计日志
- 📝 所有删除操作都会被记录
- 👤 记录操作人、时间、原因
- 🔒 仅管理员可查看日志

## ❓ 常见问题

### Q1: 忘记密码怎么办？
**A:** 联系系统管理员重置密码。

### Q2: 删除后数据还在吗？
**A:** 在的。删除为软删除，数据不会被物理删除，可以恢复。

### Q3: 谁能查看删除日志？
**A:** 只有管理员可以查看删除日志。

### Q4: 已结算的记录能删除吗？
**A:** 可以，但建议谨慎操作，可能影响财务对账。

### Q5: 删除后影响统计报表吗？
**A:** 默认统计不包含已删除记录。如需包含，使用 `include_deleted=true`。
