### 3. API 集成

#### 3.1 使用的 API 端点

| API | 方法 | 用途 |
|-----|------|------|
| `/api/unified/account/{user_id}` | GET | 获取用户账户 |
| `/api/unified/user/{user_id}/balance` | GET | 获取用户余额 |
| `/api/unified/wallet/balance` | POST | 调整余额（充值/授信） |
| `/api/unified/pickup/calculate` | POST | 计算领取详情 |
| `/api/unified/pickup/record` | POST | 记录领取 |
| `/api/unified/transactions/{user_id}` | GET | 获取交易记录 |
| `/api/unified/promotion-config/{product_id}` | GET | 获取优惠配置 |
| `/api/unified/report/financial` | GET | 财务报表 |

#### 3.2 前端数据流

```
用户操作 → Vue 组件 → API 调用 → 后端服务 → 数据库
              ↓                           ↓
          状态更新 ← 响应数据 ← 业务逻辑 ← 返回结果
              ↓
          界面渲染
```

## 🔧 技术实现

### 1. Vue 3 Composition

```javascript
const { createApp } = Vue;

createApp({
    data() {
        return {
            currentUser: null,
            products: [],
            wallets: {},
            activeTab: 'account'
        };
    },
    
    computed: {
        totalBalance() {
            return this.products.reduce((sum, p) => 
                sum + (p.balance?.prepaid_available || 0) + 
                      (p.balance?.credit_available || 0), 0
            );
        }
    },
    
    async mounted() {
        await Promise.all([
            this.checkAuth(),
            this.loadBalance(),
            this.loadProducts()
        ]);
    }
}).mount('#app');
```

### 2. API 调用封装

```javascript
async loadBalance() {
    const token = localStorage.getItem('token');
    const res = await fetch(`${API_BASE}/api/unified/user/${this.currentUser.id}/balance`, {
        headers: { 'Authorization': `Bearer ${token}` }
    });
    if (res.ok) {
        const data = await res.json();
        this.products = data.products || [];
    }
}
```

### 3. 实时计算

```javascript
async calculatePickupDetail() {
    const res = await fetch(`${API_BASE}/api/unified/pickup/calculate`, {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${token}`
        },
        body: JSON.stringify({
            user_id: this.currentUser.id,
            product_id: this.selectedProduct.product_id,
            quantity: this.pickupForm.quantity
        })
    });
    
    if (res.ok) {
        this.pickupDetail = await res.json();
    }
}
```

## 🚀 使用指南

### 管理员操作流程

#### 1. 为用户充值

```
1. 登录管理后台 → 点击"统一账户"
2. 在用户列表找到目标用户 → 点击"充值/授信"
3. 选择操作类型：

```
1. 进入"财务报表"标签页
2. 选择统计周期
3. 点击"查询"
4. 查看：
   - 总领取量
   - 预付收入
   - 信用应收
   - 分类统计
```

#### 1. 查看余额

```
1. 登录用户首页
2. 点击"我的余额"快捷入口
