---

### 问题 2: API 返回 404

**错误**: `/api/unified/pickup` 不存在

**解决**:
1. 检查路由是否注册
2. 重启 FastAPI 服务
3. 清除 Python 缓存:
   ```bash
   find . -name "*.pyc" -delete
   ```

---

### 问题 3: 前端不显示优惠信息

**原因**: 产品未配置优惠活动

**解决**:
```sql
-- 检查优惠配置
SELECT * FROM promotion_configs 
WHERE product_id = 1 AND is_active = 1;

-- 如果没有记录，添加配置
INSERT INTO promotion_configs (
    product_id, promotion_type, 
    trigger_qty, gift_qty, is_active
) VALUES (1, 'prepaid', 10, 1, 1);
```

---

### 问题 4: 前端显示异常

**原因**: 浏览器缓存

**解决**:
- Chrome: Ctrl+Shift+Delete 清除缓存
- 或使用无痕模式打开
