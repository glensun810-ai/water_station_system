# Draft 文件夹

此文件夹存放项目开发过程中的临时文档、报告、计划等非运行必需文件。

## 目录结构

```
Draft/
├── docs/                    # 文档类文件
│   ├── *.md                 # 根目录移动的过程文档
│   ├── *.txt                # 报告文件
│   ├── Requirements/        # 需求规划文档
│   ├── Service_WaterManage/ # 水站服务相关文档
│   └── Service_MeetingRoom/ # 会议室服务相关文档
├── scripts/                 # 冗余/临时脚本
│   ├── deploy-*.sh          # 旧版部署脚本
│   └── *.py                 # 临时执行脚本
└── archive/                 # 归档文件
    └── *.backup             # 备份文件
```

## 说明

这些文件保留用于：
- 历史记录参考
- 问题排查参考
- 开发过程追溯

**注意**：这些文件不是系统运行的必需文件，可根据需要随时删除。
