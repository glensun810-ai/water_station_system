> **修复日期**：2026-03-31

> **状态**：✅ API 已修复 / ⚠️ 页面访问待优化

---

## 一、问题描述

### 1.1 故障现象

访问水站管理后台时报错：
- 用户端：http://localhost:8000 → `{"detail":"Not Found"}`
- 管理后台：http://localhost:8000/admin.html → `{"detail":"Not Found"}`
- **登录 API**：`POST /api/auth/login` → ✅ 200 OK（修复后）

### 1.2 根本原因

1. **静态文件挂载顺序错误**
   - `app.mount()` 在 API 路由**之前**执行
   - 导致所有请求被 StaticFiles 处理

2. **前端目录路径错误**
   - `FRONTEND_DIR` 使用了相对路径
   - 运行时目录可能不同

3. **StaticFiles 配置问题**
   - `html=True` 参数只对目录请求有效
   - 直接请求文件（如 `/admin.html`）无效

---

## 二、修复方案

### 2.1 修复静态文件挂载位置

**文件**：`Service_WaterManage/backend/main.py`

**修改前**（第 786 行）：
```python
app = FastAPI(...)
app.mount("/", StaticFiles(directory=FRONTEND_DIR, html=True), name="static")  # ❌ 在 API 之前
# ... API 路由定义
```

**修改后**（第 4645 行）：
```python
# ... API 路由定义
app.mount("/static", StaticFiles(directory=FRONTEND_DIR), name="static")  # ✅ 在 API 之后

@app.get("/")
def root():
    return FileResponse(os.path.join(FRONTEND_DIR, "index.html"))

@app.get("/admin.html")
def admin():
    return FileResponse(os.path.join(FRONTEND_DIR, "admin.html"))
```

### 2.2 修复路径计算

**修改前**：
```python
FRONTEND_DIR = os.path.join(os.path.dirname(os.path.dirname(__file__)), "frontend")
```

**修改后**：
```python
FRONTEND_DIR = os.path.join(os.path.dirname(os.path.dirname(os.path.abspath(__file__))), "frontend")
```

### 2.3 重置管理员密码

```python
from main import SessionLocal, hash_password

    "UPDATE users SET password_hash = :hash WHERE name = 'admin'",
```

## 七、技术总结

### 7.1 FastAPI 路由优先级

1. **显式路由优先于 Mount**
2. **后注册的路由可能被覆盖**
3. **建议使用 `app.include_router()` 而非多个 `app.mount()`**

### 7.2 StaticFiles 最佳实践

```python
# 推荐方式
app.mount("/static", StaticFiles(directory="static"), name="static")

@app.get("/")
def root():
    return FileResponse("static/index.html")
```
