# 统一结算管理模块设计方案

## 一、项目背景与需求分析

### 1.1 当前系统现状

**服务类型划分：**
- 💧 **水站服务**：办公室领水记录，按办公室维度结算
- 🏢 **空间服务**：会议室、会场、大屏、展位、VIP餐厅等空间资源预约，按预约记录维度结算

**现有结算页面：**
| 页面位置 | 路径 | 功能定位 | 状态 |
|---------|-----|---------|-----|
| 水站管理-结算中心 | `/portal/admin/water/settlement_management.html` | 水站服务独立结算管理 | ✅ 已实现 |
| 空间服务-结算管理 | `/portal/admin/space/settlements.html` | 空间服务独立结算管理 | ✅ 已实现 |
| 系统管理-结算管理 | `/portal/admin/settlements.html` | Portal统一结算管理（汇总） | ⚠️ 功能不完整 |

**现有API接口：**
| 服务 | API路径 | 功能 |
|-----|--------|-----|
| 水站 | `/api/v1/water/settlements` | 水站结算记录列表 |
| 水站 | `/api/v1/water/settlements/summary` | 水站结算统计汇总 |
| 水站 | `/api/v1/water/settlement/apply` | 批量申请结算 |
| 水站 | `/api/v1/water/settlement/{id}/confirm` | 确认收款 |
| 空间 | `/api/v2/space/bookings` | 预约列表（筛选已完成用于结算） |
| 空间 | `/api/v2/space/bookings/{id}/settle` | 确认结算 |

### 1.2 核心需求

1. **保留各自独立的结算管理页面**：不影响水站和空间服务现有功能
2. **增强Portal统一结算管理页面**：提供跨服务的汇总视图
3. **统一查看管理入口**：在系统管理中提供一站式结算概览
4. **扩展性设计**：支持后续新服务类型接入

### 1.3 设计原则

- **不破坏现有功能**：保持水站和空间服务各自结算页面独立运作
- **只聚合不替代**：统一结算页面仅做汇总展示和快捷操作入口
- **复用现有API**：优先调用现有水站和空间API，不新建重复接口
- **渐进式增强**：统一结算页面可分阶段升级功能

### 2.3 功能详细设计

#### A. 汇总统计卡片区（四大指标）

| 卡片名称 | 数据来源 | 计算方式 | 点击交互 |
|---------|---------|---------|---------|
| **待确认收款** | 水站API + 空间API | 水站pending/applied + 空间completed(未结算) | 点击筛选待确认 |
| **本月已结算** | 水站API + 空间API | 本月settled/confirmed汇总 | 点击筛选已结算 |
| **本月收入** | 水站API + 空间API | 本月所有结算金额总和 | 无交互（纯展示） |
| **累计收入** | 水站API + 空间API | 历史所有已结算金额总和 | 无交互（纯展示） |

## 三、技术实现方案

### 3.1 前端实现（portal/admin/settlements.html）

**核心代码结构：**
```javascript
const { createApp, ref, computed, onMounted } = Vue;

createApp({
    setup() {
        // ========== 状态定义 ==========
        const loading = ref(false);
        const activeServiceFilter = ref('all'); // all/water/space
        const activeStatusFilter = ref('all');  // all/pending/confirmed/settled
        const activeMonthFilter = ref('current');
        const searchQuery = ref('');
        
        // ========== 数据存储 ==========
        const waterSummary = ref({});
        const spaceSummary = ref({});
        const waterRecords = ref([]);
        const spaceRecords = ref([]);
        
        // ========== 汇总计算 ==========
        const totalStats = computed(() => {
            return {
                pendingAmount: (waterSummary.value.pending_amount || 0) + 
                               (spaceSummary.value.pending_amount || 0),
                pendingCount: (waterSummary.value.pending_count || 0) + 
                              (spaceSummary.value.pending_count || 0),
                settledAmount: (waterSummary.value.settled_amount || 0) + 
                               (spaceSummary.value.settled_amount || 0),
                settledCount: (waterSummary.value.settled_count || 0) + 
                              (spaceSummary.value.settled_count || 0),
                monthlyRevenue: (waterSummary.value.monthly_revenue || 0) + 
                                (spaceSummary.value.monthly_revenue || 0),
                totalRevenue: (waterSummary.value.total_revenue || 0) + 
                              (spaceSummary.value.total_revenue || 0)
            };
        });
        
        // ========== 聚合列表 ==========
        const mergedRecords = computed(() => {
            let records = [];
            
            // 添加水站记录
            waterRecords.value.forEach(r => {
                records.push({
                    id: `water_${r.id}`,
                    serviceType: 'water',
                    serviceName: '水站服务',
                    serviceIcon: '💧',
                    recordNo: r.id,
                    payer: r.office_name,
                    amount: r.total_amount,
                    status: r.settlement_status,
                    createdAt: r.created_at,
                    detailUrl: `/portal/admin/water/settlement_management.html?id=${r.id}`
                });
            });
            
            // 添加空间记录
            spaceRecords.value.forEach(b => {
                if (b.status === 'completed' || b.status === 'settled') {
                    records.push({
                        id: `space_${b.id}`,
                        serviceType: 'space',
                        serviceName: '空间服务',
                        serviceIcon: '🏢',
                        recordNo: b.booking_no,
                        payer: b.department || b.user_name,
                        amount: b.total_fee,
                        status: b.status === 'settled' ? 'settled' : 'pending',
                        createdAt: b.created_at,
                        detailUrl: `/portal/admin/space/settlements.html?id=${b.id}`
                    });
                }
            });
            
            // 应用筛选
            if (activeServiceFilter.value !== 'all') {
                records = records.filter(r => r.serviceType === activeServiceFilter.value);
            }
            if (activeStatusFilter.value !== 'all') {
                records = records.filter(r => r.status === activeStatusFilter.value);
            }
            if (searchQuery.value) {
                const query = searchQuery.value.toLowerCase();
                records = records.filter(r => 
                    r.payer.toLowerCase().includes(query) ||
                    r.recordNo.toLowerCase().includes(query)
                );
            }
            
            // 排序：按时间倒序
            records.sort((a, b) => new Date(b.createdAt) - new Date(a.createdAt));
            
            return records;
        });
        
        // ========== 数据加载 ==========
        const loadAllData = async () => {
            loading.value = true;
            
            try {
                const token = localStorage.getItem('token');
                
                // 并行加载水站和空间数据
                const [waterRes, spaceRes] = await Promise.all([
                    fetch('/api/v1/water/settlements/summary', {
                        headers: { 'Authorization': `Bearer ${token}` }
                    }),
                    fetch('/api/v2/space/bookings?limit=100', {
                        headers: { 'Authorization': `Bearer ${token}` }
                    })
                ]);
                
                if (waterRes.ok) {
                    const waterData = await waterRes.json();
                    waterSummary.value = waterData.data || waterData;
                    waterRecords.value = waterData.records || [];
                }
                
                if (spaceRes.ok) {
                    const spaceData = await spaceRes.json();
                    const bookings = spaceData.data?.items || [];
                    // 计算空间结算汇总
                    spaceRecords.value = bookings;
                    spaceSummary.value = calculateSpaceSummary(bookings);
                }
                
            } catch (error) {
                console.error('加载结算数据失败:', error);
            } finally {
                loading.value = false;
            }
        };
        
        // 空间结算汇总计算
        const calculateSpaceSummary = (bookings) => {
            const completedBookings = bookings.filter(b => 
                b.status === 'completed' || b.status === 'settled'
            );
            
            const pendingBookings = completedBookings.filter(b => b.status === 'completed');
            const settledBookings = completedBookings.filter(b => b.status === 'settled');
            
            return {
                pending_amount: pendingBookings.reduce((sum, b) => sum + (b.total_fee || 0), 0),
                pending_count: pendingBookings.length,
                settled_amount: settledBookings.reduce((sum, b) => sum + (b.total_fee || 0), 0),
                settled_count: settledBookings.length,
                monthly_revenue: settledBookings.filter(b => {
                    const settledMonth = new Date(b.settled_at).getMonth();
                    return settledMonth === new Date().getMonth();
                }).reduce((sum, b) => sum + (b.total_fee || 0), 0),
                total_revenue: settledBookings.reduce((sum, b) => sum + (b.total_fee || 0), 0)
            };
        };
        
        onMounted(() => {
            loadAllData();
        });
        
        return {
            loading,
            activeServiceFilter,
            activeStatusFilter,
            searchQuery,
            waterSummary,
            spaceSummary,
            totalStats,
            mergedRecords,
            loadAllData
        };
    }
}).mount('#app');
```

### 4.2 不影响现有功能的保障措施

1. **独立页面独立API**：统一结算页面独立调用，不修改水站和空间现有API
2. **只读聚合展示**：统一页面仅做数据展示，不直接操作数据
3. **跳转式操作**：所有操作通过跳转到对应服务页面完成
4. **并行开发**：统一结算页面作为新模块开发，与现有模块解耦
