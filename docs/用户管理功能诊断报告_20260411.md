## 📋 问题概述

### 1. 前端URL构造错误
**问题描述**: 
- 前端页面在多处使用了错误的模板字符串拼接方式
- 将 `window.buildApiUrl()` 函数包裹在字符串中，导致API调用失败

**错误示例**:
```javascript
// ❌ 错误写法
fetch(`window.buildApiUrl('/system/users/' + user.id)`, {...})

// ✅ 正确写法  
fetch(window.buildApiUrl('/system/users/' + user.id), {...})
```

**影响范围**:
- 第1151行: toggleStatus函数
- 第1181行: promoteToSuperAdmin函数  
- 第1239行: saveUser函数
- 第1278行: resetPassword函数
- 第1293行: setAdmin函数
- 第1311行: setOfficeAdmin函数
- 第1325行: deleteUser函数
- 第1333行: batchUpdateStatus函数
- 第1344行: batchAssignOffice函数
- 第1356行: batchDelete函数（URL拼接错误）

**修复方案**: 
已将所有模板字符串错误修复为正确的函数调用方式

**缺失表清单**:

- `login_attempts` - 登录尝试记录
- `account_lockouts` - 账户锁定记录
- `user_sessions` - 用户会话管理

**修复方案**:
已创建所有安全相关的数据库表，包含完整字段结构

## 📝 使用说明

### 1. 管理员登录
```
URL: http://127.0.0.1:8008/portal/admin/users.html
用户名: admin
密码: 123456
角色: 超级管理员
```

### 2. 创建用户
1. 点击"创建用户"按钮
2. 输入用户名（支持中文、字母、数字、下划线）
3. 设置密码（默认123456）
4. 选择角色：
   - 超级管理员：系统最高权限
   - 系统管理员：管理用户和系统
   - 办公室管理员：管理特定办公室
   - 普通用户：使用服务的用户
5. 分配办公室（办公室管理员必须分配）
6. 点击"创建用户"

### 3. 重置密码
1. 点击用户操作菜单"更多"
2. 选择"重置密码"
3. 输入新密码（建议首次登录后修改）
4. 系统会自动：
   - 撤销该用户所有Token
   - 清理所有活跃会话
   - 记录密码修改日志

### 4. 批量操作
- 批量启用/停用：选择多个用户后点击对应按钮
- 批量分配办公室：选择用户后点击"分配办公室"
- 批量删除：仅超级管理员可操作
