
### 4. ✅ 统一退出跳转
- **所有角色退出** → `/portal/admin/login.html`
- **简单清晰，无歧义**

- GlobalHeader.js

- membership.html
- change-password.html

## 🔧 技术实现细节

### 智能跳转逻辑
```javascript
const getRedirectUrl = (role) => {
    // 优先级1: redirect参数
    const redirect = urlParams.get('redirect');
    if (redirect) return redirect;
    
    // 优先级2: 来源页面记忆
    const savedRedirect = sessionStorage.getItem('login_redirect');
    if (savedRedirect) return savedRedirect;
    
    // 优先级3: 角色驱动
    if (['super_admin', 'admin', 'office_admin'].includes(role)) {
        return '/portal/admin/index.html';
    } else {
        return '/portal/index.html';
    }
};
```

### 角色识别
```javascript
// 所有角色都可以登录
const userInfo = {
    id: data.user_info.user_id,
    name: data.user_info.username,
    role: data.user_info.role,
    roleName: data.user_info.role_name,
    department: data.user_info.department || '',
    avatar: getAvatarEmoji(data.user_info.role),
    is_admin: ['super_admin', 'admin', 'office_admin'].includes(data.user_info.role)
};
```
