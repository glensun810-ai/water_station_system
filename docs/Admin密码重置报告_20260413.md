# Admin账号密码重置报告

**日期:** 2026-04-13  
**问题:** 用户使用admin/admin123登录提示密码错误  
**状态:** ✅ 已修复

---

## 问题原因

1. **密码Hash不匹配** - 数据库中的密码Hash与admin123不匹配
2. **账号被锁定** - 由于之前的登录失败尝试过多，账号被锁定30分钟

---

## 解决方案

### 1. 重置密码为admin123

```python
admin.password_hash = hash_password('admin123')
```

### 2. 清除账号锁定

```sql
DELETE FROM account_lockouts WHERE username='admin';
DELETE FROM login_attempts WHERE username='admin';
```
