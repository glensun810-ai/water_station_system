## 🔧 后端API依赖（已实现）

### 阶段1 API
- `GET /api/v2/space/types` - 获取空间类型列表
- `GET /api/v2/space/resources?type_id=X` - 获取类型资源列表
- `POST /api/v2/space/resources` - 创建新资源
- `PUT /api/v2/space/resources/{id}` - 更新资源
- `DELETE /api/v2/space/resources/{id}` - 删除资源

### 阶段2 API
- `GET /api/v2/space/resources/statistics/overview` - 获取统计概览
- `POST /api/v2/space/resources/batch/activate` - 批量激活/停用
- `POST /api/v2/space/resources/batch/delete` - 批量删除
- `GET /api/v2/space/resources/export/csv` - 导出CSV

### 阶段3 API
- `GET /api/v2/space/resources/{id}/usage-statistics` - 使用统计
- `POST /api/v2/space/resources/batch/import` - 批量导入
