# my_bookings.html 500错误修复报告

## 问题定位

**错误现象**：
```
GET http://127.0.0.1:8008/api/v1/meeting/bookings?is_deleted=0 500 (Internal Server Error)
AttributeError: 'NoneType' object has no attribute 'role'
```

**根本原因**：
1. API端点 `get_bookings` 使用 `get_current_user` 依赖，该依赖返回 `Optional[User]`
2. 当用户未登录或token无效时，`current_user` 为 `None`
3. 代码直接访问 `current_user.role` 导致 AttributeError
