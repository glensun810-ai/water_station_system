## 五、使用指南

### 5.1 快速开始

1. 引入设计系统样式：
```html
<link rel="stylesheet" href="../assets/css/design-system.css">
```

2. 引入Vue 3：
```html
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```

3. 引入组件并使用：
```html
<template>
  <OfficeSelector v-model="selectedOffice" @change="handleChange" />
</template>
```

详细使用方法请参考：`portal/docs/README.md`
