## 问题现象
用户访问Phase 3的支付相关页面时，浏览器显示空白页面，无法看到任何内容。

---

## 根本原因
**GlobalHeader组件加载失败导致Vue应用初始化失败**

技术细节：
- 页面依赖GlobalHeader组件
- 组件加载时机与Vue初始化不同步
- 导致整个Vue应用无法挂载，页面空白

---

## 修复方案
**移除GlobalHeader组件依赖，简化页面结构**

### 修复内容
1. 移除GlobalHeader相关的HTML标签
2. 移除GlobalHeader的脚本和样式引用
3. 移除Vue组件注册
4. 移除相关的props和事件处理
5. 保留核心业务功能

### 修复文件
- ✓ membership-plans.html
- ✓ payment.html
- ✓ orders.html
- ✓ invoice-apply.html
- ✓ invoices.html

### 备份文件
所有原文件已备份为`.backup`后缀
