## 💡 使用指南

### 访问地址

**统一管理后台**: http://localhost:8000/portal/admin/login.html

**登录账号**:
- 用户名: admin
- 密码: admin123

### 操作流程

**新增领水记录**:
1. 登录管理后台
2. 点击左侧"领水记录"
3. 点击"新增领水记录"按钮
4. 填写表单信息
5. 点击"提交"

**编辑领水记录**:
1. 在列表中找到要编辑的记录
2. 点击"编辑"按钮
3. 修改信息
4. 点击"提交"

**删除领水记录**:
1. 在列表中找到要删除的记录
2. 点击"删除"按钮
3. 确认删除操作
