## 一、页面功能概述

### 1.1 页面定位
`http://127.0.0.1:8008/water/index.html` 是水站服务的核心业务页面，为用户提供完整的领水业务流程，包括：
- **办公室选择与设置**
- **产品选购与数量管理**
- **领水申请提交**
- **领水记录查看与管理**
- **结算申请与付款管理**

### 1.2 用户角色
页面支持多种用户角色，并根据角色提供不同的功能和权限：
- **普通用户**：只能查看和使用自己所属办公室
- **办公室管理员**：可以管理所属办公室的领水记录和结算
- **系统管理员/超级管理员**：可以查看所有办公室，管理全局业务

**买N赠M优惠计算**（Line 1062-1078）：
```javascript
totalGiftQty() {
    return this.products.reduce((sum, p) => {
        if (!this.cart[p.id]) return sum;
        const qty = this.cart[p.id];
        
        // 判断是否满足优惠条件
        if (p.promo_threshold && p.promo_gift && qty >= p.promo_threshold) {
            const cycle = p.promo_threshold + p.promo_gift;
            const cycles = Math.floor(qty / cycle);
            const remainder = qty % cycle;
            
            let gift = cycles * p.promo_gift;
            
            // 处理余数：如果余数大于阈值，还有额外赠送
            if (remainder > p.promo_threshold) {
                gift += Math.min(remainder - p.promo_threshold, p.promo_gift);
            }
            
            sum += gift;
        }
        return sum;
    }, 0);
}
```

**总金额计算**（Line 1079-1098）：
```javascript
totalAmount() {
    // 原价总额
    const original = this.products.reduce((sum, p) => {
        if (!this.cart[p.id]) return sum;
        return sum + this.cart[p.id] * p.price;
    }, 0);
    
    // 折扣金额（赠送部分的金额）
    const discount = this.products.reduce((sum, p) => {
        if (!this.cart[p.id] || !p.promo_threshold || !p.promo_gift) return sum;
        const qty = this.cart[p.id];
        if (qty < p.promo_threshold) return sum;
        
        // 计算赠送数量
        const cycle = p.promo_threshold + p.promo_gift;
        const cycles = Math.floor(qty / cycle);
        const remainder = qty % cycle;
        let gift = cycles * p.promo_gift;
        if (remainder > p.promo_threshold) {
            gift += Math.min(remainder - p.promo_threshold, p.promo_gift);
        }
        
        return sum + gift * p.price;
    }, 0);
    
    return (original - discount).toFixed(2);
}
```

**优惠算法详解**：
- `promo_threshold`：买N桶触发优惠
- `promo_gift`：赠送M桶
- 例如：买10赠1，买15桶 → 优惠周期(10+1=11) × 1 = 11桶，余数4桶不触发赠送
- 例如：买10赠1，买13桶 → 优惠周期1次(11桶)，余数2桶，额外赠送min(2-10,1)=0，总共赠送1桶

### 4.1 数据加载流程（Line 1290-1349）

```javascript
async loadData() {
    // 1. 获取用户角色和办公室信息
    const userRole = localStorage.getItem('user_role');
    const storedUserOffices = localStorage.getItem('user_offices');
    const hasSkippedGuide = localStorage.getItem('has_skipped_first_time_guide');
    
    // 2. 初始化用户办公室列表
    if (storedUserOffices) {
        this.userOffices = JSON.parse(storedUserOffices);
    }
    
    // 3. 首次使用引导逻辑
    if (!storedUserOffices && !hasSkippedGuide && !this.isAdmin) {
        this.showFirstTimeGuide = true;
    }
    
    try {
        // 4. 并行加载办公室和产品数据
        const [officesRes, productsRes] = await Promise.all([
            fetch(API_BASE + '/water/offices?is_active=true'),
            fetch(API_BASE + '/water/products?active_only=true')
        ]);
        
        if (officesRes.ok) this.offices = await officesRes.json();
        
        if (productsRes.ok) {
            const productsData = await productsRes.json();
            if (productsData && productsData.length > 0) {
                this.products = productsData;
                // 初始化购物车：所有产品数量为0
                this.products.forEach(p => this.cart[p.id] = 0);
            }
        }
        
        // 5. 设置默认办公室
        if (this.isAdmin) {
            // 管理员：默认选择第一个办公室
            if (this.offices.length > 0 && !this.form.officeId) {
                this.form.officeId = this.offices[0].id;
                this.settlementOfficeId = this.offices[0].id;
            }
        } else {
            // 普通用户：只能使用自己的办公室（API已过滤）
            if (this.offices.length > 0 && !this.form.officeId) {
                this.form.officeId = this.offices[0].id;
                this.settlementOfficeId = this.offices[0].id;
                
                // 自动设置领水人为办公室负责人
                if (this.offices[0].leader_name) {
                    this.form.personName = this.offices[0].leader_name;
                    this.form.personId = '';
                    this.useManualPerson = true;
                }
            }
        }
    } catch (e) {
        console.error('加载数据失败:', e);
        console.error('错误详情:', e.message);
    }
    
    // 6. 加载支付二维码
    this.loadPaymentQr();
}
```

**数据加载策略**：
- 并行加载（Promise.all）提升性能
- 根据用户角色设置默认办公室
- 普通用户自动填充领水人信息
- 失败时详细错误日志便于排查

### 4.2 领水提交流程（Line 1656-1668）

```javascript
async submitPickup() {
    // 1. 表单验证
    if (!this.canSubmit) {
        this.showError('请填写完整信息');
        return;
    }
    
    const items = Object.entries(this.cart).filter(([k, v]) => v > 0);
    if (items.length === 0) {
        this.showError('请选择产品数量');
        return;
    }
    
    // 2. 获取领水人姓名
    const personName = this.form.personId 
        ? (this.officeUsers.find(u => u.id === this.form.personId)?.name || '') 
        : this.form.personName;
    
    try {
        // 3. 提交领水记录（每个产品一条记录）
        for (const [productId, qty] of items) {
            const res = await fetch(API_BASE + '/water/pickup', {
                method: 'POST',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({
                    office_id: this.form.officeId,
                    product_id: parseInt(productId),
                    quantity: qty,
                    pickup_person: personName,
                    pickup_person_id: this.form.personId || null,
                    pickup_time: this.form.pickupTime 
                        ? new Date(this.form.pickupTime).toISOString() 
                        : null
                })
            });
            
            if (!res.ok) {
                const e = await res.json();
                this.showError(e.detail || '提交失败');
                return;
            }
        }
        
        // 4. 成功反馈和重置
        this.successMessage = '提交成功！';
        this.showSuccess = true;
        this.resetForm();
        await this.loadData();
        
    } catch (e) {
        this.showError('网络错误');
    }
}
```

**提交逻辑详解**：
- 每个产品提交一条独立的领水记录
- 支持指定领水时间（可选）
- 领水人可以是用户列表中的用户，也可以手动输入
- 成功后重置表单并刷新数据

**交互细节**：

   - 显示优惠信息（买N赠M）

   - 实时计算总金额和赠送数量

   - 可手动输入其他领水人

   - 可选择其他时间
   - 支持补录历史领水记录

5. **提交反馈**：
   - 成功：显示成功消息，重置表单
   - 失败：显示错误详情，保留表单数据

### 7.3 结算流程

2. **结算申请**：

   - 生成结算单号
   - 显示结算详情

3. **结算详情**：
   - 展开显示包含的所有领水记录
   - 显示总金额、总数量
   - 显示申请时间、确认时间
   - 显示支付二维码
