## 一、项目概览

### 1.1 项目背景

AI产业集群空间服务系统包含水站管理、会议室预定等多个服务模块，前端代码存在以下问题：
- 模块边界不清，代码冗余严重
- 缺乏统一的组件库和设计系统
- 无统一的服务入口
- 管理后台分散，维护成本高

### 1.2 项目目标

1. 建立统一的组件库和设计系统
2. 开发统一的服务入口（Portal首页）
3. 整合管理后台，统一入口和权限管理
4. 提升开发效率和用户体验

## 九、使用指南

### 9.1 快速开始

1. **访问Portal首页**
   - URL: `http://localhost:8000/`
   - 查看服务入口和用户信息

2. **访问管理后台**
   - URL: `http://localhost:8000/admin/login.html`
   - 用户名: `admin`
   - 密码: `admin123`

### 9.2 组件使用

参考文档: `portal/docs/README.md`

示例代码:
```html
<!-- 引入设计系统 -->
<link rel="stylesheet" href="../assets/css/design-system.css">

<!-- 使用组件 -->
<OfficeSelector v-model="selectedOffice" @change="handleChange" />
<StatsCard title="今日预约" :value="23" icon="📅" color="primary" />
```
