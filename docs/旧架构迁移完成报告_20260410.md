## 六、回滚方案

### 6.1 快速回滚

如果新架构出现严重问题，可以快速回滚到旧架构：

```bash
#!/bin/bash
# rollback.sh

echo "开始回滚到旧架构..."

# 1. 停止新架构服务
pkill -f "apps/main.py"

# 2. 恢复旧架构目录
cp -r Backup/old_architecture_20260410_200812/Service_WaterManage ./
cp -r Backup/old_architecture_20260410_200812/Service_MeetingRoom ./
cp -r Backup/old_architecture_20260410_200812/Service_Dining ./

# 3. 启动旧架构服务
cd Service_WaterManage/backend
PORT=8008 python main.py &

cd ../../Service_MeetingRoom/backend
PORT=8009 python main.py &

echo "回滚完成！"
echo "水站服务: http://127.0.0.1:8008/water/index.html"
echo "会议室服务: http://127.0.0.1:8009/meeting/index.html"
```
