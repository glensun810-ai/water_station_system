## 完整修复方案

### 修复1：添加sortedOffices计算属性

```javascript
// Line 1035-1051 (apps/water/frontend/index.html)
sortedOffices() {
    // 按房间号排序（支持数字和字符串）
    return this.offices.sort((a, b) => {
        const roomA = a.room_number || '';
        const roomB = b.room_number || '';
        
        // 尝试按数字排序
        const numA = parseInt(roomA);
        const numB = parseInt(roomB);
        
        if (!isNaN(numA) && !isNaN(numB)) {
            return numA - numB;  // 数字排序
        } else {
            return roomA.localeCompare(roomB, 'zh');  // 字符串排序
        }
    });
}
```

**排序逻辑**：
- 尝试将room_number转换为数字
- 如果都是数字，按数字排序（001 < 002 < 003）
- 如果不是数字，按字符串排序（支持中文）
- 例如：001总经理、002副总经理、003北大博士后站...

### 修复2：修改displayOffices逻辑

```javascript
// Line 1053-1080 (apps/water/frontend/index.html)
displayOffices() {
    const userRole = localStorage.getItem('user_role');
    
    // 超级管理员/管理员：显示所有办公室（按房间号排序）
    if (userRole === 'admin' || userRole === 'super_admin') {
        // 如果选择了特定标签，按标签过滤
        if (this.officeTab === 'common') {
            return this.sortedOffices.filter(o => o.is_common === 1);
        } else if (this.officeTab === 'uncommon') {
            return this.sortedOffices.filter(o => o.is_common !== 1);
        } else {
            // 默认显示所有办公室（全部）
            return this.sortedOffices;
        }
    }
    
    // 普通用户/办公室管理员：显示用户所属办公室
    if (this.userOffices.length > 0) {
        const userOfficeIds = this.userOffices.map(uo => uo.id);
        return this.sortedOffices.filter(o => userOfficeIds.includes(o.id));
    }
    
    // 如果 userOffices 为空，显示后端返回的 offices（后端已过滤权限）
    return this.sortedOffices;
}
```

**逻辑优化**：
- 使用sortedOffices（已排序）
- officeTab='all'：显示所有21个办公室
- officeTab='common'：显示17个常用办公室
- officeTab='uncommon'：显示4个不常用办公室
- 普通用户：显示所属办公室
