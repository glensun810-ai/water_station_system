# 水站领水登记权限控制实现报告

## 📋 需求分析

### 用户需求
管理员打开水站领水登记时，需要：
- ✓ 看到所有办公室
- ✓ 可以选择任意一个办公室进行登记领水

普通用户仅：
- ✓ 看到自己所属的办公室
- ✓ 仅对自己办公室可以登记领水

## ✅ 实现方案

### 1. 后端API权限控制

#### 办公室列表API（`GET /api/v1/water/offices`）

**文件**：`apps/api/v1/water.py`

**实现逻辑**：
```python
@router.get("/offices")
def get_active_offices(
    is_active: bool = Query(True, description="是否只返回激活的办公室"),
    db: Session = Depends(get_db),
    current_user: User = Depends(get_current_user),
):
    """
    获取办公室列表（权限过滤）
    
    权限规则：
    - 管理员（admin, super_admin）：可以看到所有办公室
    - 普通用户（user, office_admin）：只能看到自己所属的办公室
    """
    
    query = db.query(Office)
    
    if is_active:
        query = query.filter(Office.is_active == True)
    
    # 权限过滤：普通用户只能看到自己所属的办公室
    if current_user.role not in ["admin", "super_admin"]:
        # 通过department字段匹配办公室名称
        if current_user.department:
            query = query.filter(Office.name == current_user.department)
        else:
            # 如果用户未设置department，返回空列表
            return []
    
    offices = query.order_by(Office.name).all()
    
    # ... 返回办公室列表
```

**关键点**：
- 管理员：返回所有办公室
- 普通用户：通过 `User.department` 字段匹配 `Office.name`，只返回匹配的办公室
- 未设置部门：返回空列表

#### 领水创建API（`POST /api/v1/water/pickup`）

**文件**：`apps/api/v1/water.py`

**实现逻辑**：
```python
@router.post("/pickup")
def create_water_pickup(
    pickup_data: dict,
    db: Session = Depends(get_db),
    current_user: User = Depends(get_current_user),
):
    """
    创建领水记录（权限验证）
    
    权限规则：
    - 管理员（admin, super_admin）：可以为任何办公室领水
    - 普通用户（user, office_admin）：只能为自己所属的办公室领水
    """
    
    office_id = pickup_data.get("office_id")
    
    # 验证办公室
    office = db.query(Office).filter(Office.id == office_id, Office.is_active == True).first()
    if not office:
        raise HTTPException(status_code=404, detail="办公室不存在或已停用")
    
    # 权限验证：普通用户只能为自己所属的办公室领水
    if current_user.role not in ["admin", "super_admin"]:
        if not current_user.department:
            raise HTTPException(
                status_code=403, 
                detail="您未设置所属办公室，请联系管理员设置"
            )
        
        if office.name != current_user.department:
            raise HTTPException(
                status_code=403,
                detail=f"您只能为所属办公室 '{current_user.department}' 进行领水登记，无权限为 '{office.name}' 登记"
            )
    
    # ... 创建领水记录
```

**关键点**：
- 管理员：可以为任何办公室创建领水记录
- 普通用户：验证 `office.name` 是否匹配 `user.department`
- 不匹配时返回 403 错误

#### 角色判断逻辑

**computed属性**：
```javascript
computed: {
    isAdmin() {
        // 从localStorage获取用户角色信息
        const userRole = localStorage.getItem('user_role');
        return userRole === 'admin' || userRole === 'super_admin';
    },
    selectedOffice() { 
        return this.offices.find(o => o.id === this.form.officeId); 
    },
    // ...
}
```

                this.form.personName = this.offices[0].leader_name;
            }
        }
    }
}

## 🔐 数据字段说明

### User模型字段

- `department`：存储用户所属的办公室名称（字符串）
- `role`：用户角色（super_admin/admin/office_admin/user）

### Office模型字段

- `name`：办公室名称（用于匹配User.department）
- `room_number`：房间号
- `leader_name`：负责人姓名

## 🎯 使用说明

### 管理员操作流程

1. 登录系统（admin/super_admin角色）
2. 打开水站领水登记页面
3. 查看所有办公室列表（常用/不常用分类）
4. 选择任意办公室

### 普通用户操作流程

1. 登录系统（user角色，需设置department）

3. 自动显示所属办公室
4. 填写领水信息并提交（仅限所属办公室）
