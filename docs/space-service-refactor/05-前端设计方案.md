## 一、前端架构总览

### 1.1 技术栈选择

| 技术项 | 选择方案 | 说明 |
|--------|---------|------|
| **框架** | Vue 3 (Composition API) | 与现有系统保持一致 |
| **样式** | TailwindCSS | 快速开发，响应式优先 |
| **状态管理** | Pinia | Vue 3官方推荐 |
| **路由** | Vue Router 4 | SPA路由管理 |
| **HTTP请求** | Axios | 统一请求拦截 |
| **UI组件** | Headless UI + 自定义组件 | 轻量灵活 |
| **图表** | ECharts | 统计图表展示 |
| **日历** | FullCalendar | 专业日历组件 |
| **表单验证** | VeeValidate | 表单验证 |
| **通知** | Toast Notifications | 操作反馈 |

### 4.1 拒绝弹窗组件（RejectDialog.vue）★新增

```vue
<template>
  <div class="reject-dialog" v-if="visible">
    <div class="dialog-overlay" @click="close"></div>
    <div class="dialog-content">
      <h3 class="dialog-title">拒绝预约申请</h3>
      
      <!-- 拒绝原因选择 -->
      <div class="reject-reasons">
        <h4>拒绝原因（必填）</h4>
        <div class="reason-options">
          <label v-for="reason in reasonOptions" :key="reason.value">
            <input type="checkbox" 
                   v-model="selectedReasons" 
                   :value="reason.value">
            <span>{{ reason.label }}</span>
          </label>
        </div>
        <textarea v-model="detailReason" 
                  placeholder="请输入详细拒绝原因..."
                  rows="4"></textarea>
      </div>
      
      <!-- 是否允许重新提交 -->
      <div class="resubmit-option">
        <h4>是否允许重新提交</h4>
        <label>
          <input type="radio" v-model="allowResubmit" value="true">
          <span>允许申请人修改后重新提交</span>
        </label>
        <label>
          <input type="radio" v-model="allowResubmit" value="false">
          <span>此申请最终拒绝</span>
        </label>
      </div>
      
      <!-- 推荐备选方案 -->
      <div class="suggest-alternatives" v-if="allowResubmit">
        <h4>推荐备选方案（可选）</h4>
        <div class="alternative-options">
          <label v-for="alt in alternatives" :key="alt.resource_id">
            <input type="checkbox" 
                   v-model="selectedAlternatives" 
                   :value="alt">
            <span>{{ alt.resource_name }} | {{ alt.available_date }}</span>
          </label>
        </div>
      </div>
      
      <!-- 操作按钮 -->
      <div class="dialog-actions">
        <button class="btn-cancel" @click="close">取消</button>
        <button class="btn-confirm" @click="confirmReject">确认拒绝</button>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue';

const props = defineProps({
  visible: Boolean,
  approvalId: Number,
  bookingId: Number
});

const emit = defineEmits(['close', 'reject']);

const reasonOptions = [
  { value: 'time_conflict', label: '时间冲突，档期已满' },
  { value: 'safety_issue', label: '方案不符合安全规范' },
  { value: 'info_incomplete', label: '预约信息不完整' },
  { value: 'budget_exceeded', label: '预算超出范围' },
  { value: 'other', label: '其他原因' }
];

const selectedReasons = ref([]);
const detailReason = ref('');
const allowResubmit = ref('true');
const selectedAlternatives = ref([]);
const alternatives = ref([]);

const confirmReject = async () => {
  if (selectedReasons.value.length === 0) {
    alert('请选择至少一个拒绝原因');
    return;
  }
  
  const rejectData = {
    approval_id: props.approvalId,
    booking_id: props.bookingId,
    rejection_reason: selectedReasons.value.join(', '),
    rejection_detail: detailReason.value,
    allow_resubmit: allowResubmit.value === 'true',
    suggest_alternatives: selectedAlternatives.value
  };
  
  emit('reject', rejectData);
  close();
};

const close = () => {
  emit('close');
};
</script>
```
