## 一、空间类型业务场景详细分析

### 1.1 会议室（Meeting Room）

**业务定位:** 小型团队协作空间

| 维度 | 详细说明 |
|------|---------|
| **目标用户** | 内部员工、外部访客、小型团队（2-20人） |
| **核心场景** | 日常会议、项目讨论、面试、培训、临时协作 |
| **预约时长** | 最小30分钟，通常1-2小时，最长半天 |
| **计价单位** | 按小时计价 |
| **定价因素** | 容量、设备配置、时间段（工作时间/非工作时间） |
| **审批需求** | 内部员工无需审批，外部访客需审批 |
| **支付方式** | 内部月结，外部预付 |
| **设备需求** | 投影仪、白板、音响、网络、空调 |
| **典型流程** | 选择会议室 → 选择时段 → 填写会议主题 → 提交 → 即时确认（内部）或审批确认（外部） |

**特殊需求:**
- 免费额度: 内部员工每月2小时免费使用
- 快速预约: 希望能在1分钟内完成预约
- 即时确认: 内部员工预约后自动确认，无需等待审批
- 设备预约: 可同时预约投影仪等设备

**痛点分析:**
1. 预约冲突频繁，需智能推荐备选时段
2. 会议临时取消，资源浪费
3. 设备故障影响会议质量
4. 无法提前查看会议室状态（是否空闲/是否干净）

### 1.2 会场/多功能厅（Venue/Event Hall）

**业务定位:** 大型活动举办空间

| 维度 | 详细说明 |
|------|---------|
| **目标用户** | 活动组织者、企业年会、发布会、培训课程 |
| **核心场景** | 企业年会、产品发布会、大型培训、行业论坛、庆典活动 |
| **预约时长** | 最小半天（4小时），通常全天（8小时），最长连续多天 |
| **计价单位** | 按半天/全天计价，不支持按小时 |
| **定价因素** | 容量、设备配置、活动类型、预订提前期 |
| **审批需求** | 必须审批，需提前至少3天预约 |
| **支付方式** | 预付定金30%，活动前付清尾款 |
| **设备需求** | 专业音响、灯光、舞台、LED大屏、直播设备、茶歇服务 |
| **典型流程** | 提交活动方案 → 审核方案可行性 → 审批通过 → 签订合同 → 支付定金 → 活动准备 → 活动执行 → 活动结算 |

**特殊需求:**
- 活动方案审核: 需提交活动流程、参会人数、安全方案
- 定金支付: 必须支付定金才能锁定档期
- 配套服务: 茶歇、餐饮、摄影、直播等增值服务
- 活动保险: 建议购买活动保险
- 播场时间: 需提前2小时进入播场
- 播场费用: 播场时间单独计费

**痛点分析:**
1. 档期冲突，热门日期难预约
2. 活动准备时间不足
3. 配套服务协调困难
4. 活动现场突发问题处理
5. 活动效果评估缺失

**定价策略示例:**
```
小型会场（50-100人）:
- 半天: 2,000元
- 全天: 3,500元
- 播场费: 500元/小时

大型会场（100-500人）:
- 半天: 5,000元
- 全天: 8,000元
- 播场费: 1,000元/小时

增值服务:
- 专业音响: +500元
- LED大屏: +1,000元
- 茶歇服务: +50元/人
- 直播服务: +2,000元
```

### 1.3 大堂大屏（Lobby Display Screen）

**业务定位:** 企业形象展示与广告投放

| 维度 | 详细说明 |
|------|---------|
| **目标用户** | 企业品牌部门、广告投放商、活动宣传部门 |
| **核心场景** | 企业宣传片播放、产品广告投放、活动宣传、节日祝福、重要通知 |
| **预约时长** | 最小1小时，通常半天或全天，最长一周 |
| **计价单位** | 按时段计价（黄金时段/普通时段） |
| **定价因素** | 时段（早高峰/午高峰/晚高峰）、内容类型、播放频次 |
| **审批需求** | 内容审核，确保内容合规 |
| **支付方式** | 预付全款 |
| **设备需求** | LED大屏、播放系统、网络 |
| **典型流程** | 提交播放内容 → 内容审核 → 审批通过 → 选择时段 → 支付费用 → 内容上架 → 播放执行 → 播放结束 |

**特殊需求:**
- 内容审核: 广告内容需审核合规性
- 播放排期: 多个内容按时间段轮播
- 效果监测: 提供播放次数、观看人数统计
- 紧急下架: 违规内容可紧急下架
- 定时播放: 支持定时播放和循环播放

**时段划分:**
```
黄金时段（高曝光）:
- 早高峰 08:00-10:00: 200元/小时
- 午高峰 11:30-13:30: 200元/小时
- 晚高峰 17:00-19:00: 200元/小时

普通时段:
- 工作时间 09:00-17:00（排除高峰）: 100元/小时
- 非工作时间 19:00-08:00: 50元/小时（夜间展示）

全天套餐:
- 8小时工作时段: 800元
- 24小时全天: 1,000元
- 一周套餐: 5,000元（7天）
```

**痛点分析:**
1. 内容审核流程慢
2. 播放时段冲突
3. 无法实时查看播放效果
4. 效果数据缺失
5. 紧急内容无法快速上架

### 1.4 大堂展位（Lobby Exhibition Booth）

**业务定位:** 产品展示与品牌推广

| 维度 | 详细说明 |
|------|---------|
| **目标用户** | 企业市场部门、产品推广团队、品牌展示 |
| **核心场景** | 新产品展示、品牌宣传、节日活动、合作推广、客户接待 |
| **预约时长** | 最小1天，通常3-5天，最长一个月 |
| **计价单位** | 按天计价，支持周套餐、月套餐 |
| **定价因素** | 展位位置（中心/边缘）、展位大小、展示类型 |
| **审批需求** | 展示方案审批，需提前至少1周预约 |
| **支付方式** | 预付定金50%，展示前付清尾款 |
| **设备需求** | 展台、展架、电源、照明、网络 |
| **典型流程** | 提交展示方案 → 方案审批 → 签订合同 → 支付定金 → 展位布置 → 展示执行 → 展示结束 → 效果评估 |

**特殊需求:**
- 展位布置: 需提前1天进入布置
- 安全检查: 展示物品需安全检查
- 清洁维护: 每日清洁维护
- 人流统计: 提供展位人流数据
- 值班人员: 提供值班人员服务（可选）

**展位类型:**
```
中心展位（高曝光）:
- 标准展位（2m×2m）: 500元/天
- 大型展位（4m×4m）: 1,000元/天

边缘展位:
- 标准展位（2m×2m）: 300元/天
- 大型展位（4m×4m）: 600元/天

套餐优惠:
- 周套餐（7天）: 标准展位 2,000元（原价3,500元）
- 月套餐（30天）: 标准展位 8,000元（原价15,000元）

增值服务:
- 值班人员: 300元/天
- 专业布置: 500元
- 清洁服务: 100元/天
```

**痛点分析:**
1. 热门展位抢订
2. 展示效果难以量化
3. 展位布置时间不足
4. 展示期间维护困难
5. 展示物品存放问题

### 1.5 VIP餐厅包厢（VIP Dining Room）

**业务定位:** 高端商务宴请空间

| 维度 | 详细说明 |
|------|---------|
| **目标用户** | VIP客户接待、商务宴请、重要会议用餐、合作伙伴接待 |
| **核心场景** | VIP客户接待、商务谈判、重要会议、合作伙伴用餐、庆典宴请 |
| **预约时长** | 按餐段预约（午餐时段/晚餐时段），约2小时 |
| **计价单位** | 按餐段计价，不按时段拆分 |
| **定价因素** | 包厢等级、餐标选择、增值服务 |
| **审批需求** | 内部员工预约需主管审批，外部客户预约需预付 |
| **支付方式** | 餐后结算（内部），预付定金（外部） |
| **设备需求** | 餐桌餐具、音响、空调、独立卫生间、私密空间 |
| **典型流程** | 选择包厢 → 选择餐段 → 选择餐标 → 填写客人信息 → 提交审批 → 审批通过 → 支付定金 → 用餐执行 → 餐后结算 |

**特殊需求:**
- 餐标选择: 不同餐标价格不同
- 客人信息: 需填写客人人数、特殊需求
- 特殊需求: 如素食、过敏提醒、生日蛋糕等
- 用餐时长: 餐段时长2小时，超时额外收费
- 服务人员: 配备专属服务员

**餐段定义:**
```
午餐时段:
- 11:30-14:00（2.5小时）

晚餐时段:
- 17:30-20:00（2.5小时）
- 20:00-22:00（夜宵时段，可选）
```

**定价策略:**
```
包厢等级:
- 标准包厢（6-8人）: 包厢费 500元/餐段
- VIP包厢（10-12人）: 包厢费 800元/餐段
- 豪华包厢（15-20人）: 包厢费 1,200元/餐段

餐标选择:
- 标准餐标: 200元/人
- VIP餐标: 300元/人
- 豪华餐标: 500元/人

总费用 = 包厢费 + (餐标 × 人数)

示例:
- VIP包厢 + 标准餐标 + 10人 = 800 + 200×10 = 2,800元
- 豪华包厢 + VIP餐标 + 15人 = 1,200 + 300×15 = 5,700元

增值服务:
- 生日蛋糕: 200元
- 专业服务团队: +300元
- 餐后茶歇: +100元
```

**痛点分析:**
1. 餐段冲突，热门时段难预约
2. 客人临时取消，费用浪费
3. 特殊需求沟通困难
4. 用餐效果难以评估
5. 费用预算不透明

### 2.2 预约状态流转模型

定义统一的预约状态机，适用于所有空间类型:

```
预约状态流转图:

[初始状态]
    ↓
创建预约 → pending_approval（待审批）
    │           ↓
    │       审批流程 → approved（已审批）→ confirmed（已确认）→ in_use（使用中）
    │           │           │                   │                   │
    │           │           │                   │                   │
    │           ↓           ↓                   ↓                   ↓
    │       rejected（已拒绝）cancelled（已取消）  completed（已完成）
    │
    ↓（无需审批）
confirmed（已确认）→ in_use → completed

状态定义:
- pending_approval: 预约已提交，等待审批
- approved: 预约已通过审批，等待支付定金/确认
- confirmed: 预约已确认，时段已锁定
- in_use: 预约正在执行（可选状态，用于签到）
- completed: 预约已完成，等待结算
- cancelled: 预约已取消
- rejected: 预约已拒绝

状态流转规则:
1. pending_approval → approved: 审批通过
2. pending_approval → rejected: 审批拒绝
3. pending_approval → cancelled: 申请人主动取消（审批前）
4. approved → confirmed: 支付定金确认（如需）或自动确认
5. approved → cancelled: 申请人主动取消（审批后）
6. confirmed → in_use: 签到入场（可选）
7. confirmed → cancelled: 申请人主动取消（确认后，需扣除定金）
8. in_use → completed: 使用结束确认
9. completed → settled: 费用结算完成（新增状态）

特殊状态补充:
- deposit_paid: 定金已支付（会场/展位/VIP餐厅）
- deposit_refunded: 定金已退回（取消后）
- settled: 结算已完成
```

### 2.3 定价策略统一模型

提取各空间类型的定价共性，构建统一定价模型:

```
定价策略模型:

1. 基础定价（Base Pricing）
   ├── 计价单位（pricing_unit）: hour / half_day / full_day / meal_session / day / week
   ├── 基础价格（base_price）: 根据空间类型和等级确定
   └── 时段定价（time_slot_pricing）: 不同时段不同价格
       ├── 工作时间价格
       ├── 非工作时间价格
       ├── 黄金时段价格（大屏特有）
       └── 餐段价格（VIP餐厅特有）

2. 容量定价（Capacity Pricing）
   ├── 容量等级（capacity_level）: small / medium / large
   └── 容量价格系数（capacity_factor）: 1.0 / 1.2 / 1.5

3. 会员定价（Member Pricing）
   ├── 会员等级（member_level）: regular / VIP / enterprise
   ├── 会员折扣率（member_discount）: 1.0 / 0.8 / 0.7
   └── 每月免费额度（free_quota）: 0 / 2小时 / 5小时

4. 时长优惠（Duration Discount）
   ├── 连续预约优惠: 预约时长越长折扣越大
   │    ├── 2-4小时: 95%价格
   │    ├── 4-8小时: 90%价格
   │    ├── 全天: 85%价格
   ├── 多天预约优惠: 连续多天折扣
   │    ├── 3-7天: 90%价格
   │    ├── 7-30天: 80%价格

5. 时段优惠（Time Discount）
   ├── 非热门时段优惠: 非黄金时段折扣
   ├── 节假日定价: 节假日价格调整（上浮或下降）
   └── 促销时段: 限时优惠时段

6. 批量优惠（Batch Discount）
   ├── 企业批量预约: 企业月度预约量达标后折扣
   └── 套餐优惠: 周套餐、月套餐折扣

7. 增值服务定价（Add-on Pricing）
   ├── 设备租赁: 投影仪、音响、LED大屏等
   ├── 服务人员: 值班人员、服务员、技术支持
   ├── 配套服务: 茶歇、餐饮、摄影、直播
   └── 布置服务: 专业布置、清洁服务

8. 定金规则（Deposit Rule）
   ├── 定金比例: 会场30%、展位50%、VIP餐厅30%
   ├── 定金锁定: 支付定金后锁定时段
   └── 定金退还规则: 
       ├── 提前7天取消: 退还100%定金
       ├── 提前3天取消: 退还50%定金
       ├── 提前1天取消: 不退定金
       ├── 使用后取消: 不退定金

定价计算公式:
最终价格 = (基础价格 × 时段系数 × 容量系数 × 会员折扣 × 时长优惠 × 批量优惠 + 增值服务费用) × 定金比例（如需）

示例计算:
会议室预约（VIP会员，4小时）:
= (50元/小时 × 4小时 × 1.0 × 0.8 × 0.95 × 1.0) + 0
= 152元

会场预约（全天，无会员）:
= (3500元/天 × 1.0 × 1.0 × 1.0 × 0.85 × 1.0 + 茶歇50元×100人 + LED大屏1000元) × 30%定金
= (2975元 + 5000元 + 1000元) × 30%
= 2692.5元定金
```

## 四、业务规则汇总

### 4.1 预约规则

| 规则类型 | 规则详情 |
|---------|---------|
| **预约时间限制** | 会议室: 最小30分钟，最大半天<br>会场: 最小半天，最大连续3天<br>大屏: 最小1小时，最大一周<br>展位: 最小1天，最大一个月<br>VIP餐厅: 按餐段预约，固定2.5小时 |
| **预约提前期** | 会议室: 可当天预约<br>会场: 至少提前3天<br>大屏: 至少提前1天<br>展位: 至少提前1周<br>VIP餐厅: 至少提前1天 |
| **预约冲突检测** | 同一空间同一时间段不可重复预约<br>检测范围: pending_approval + approved + confirmed + in_use |
| **预约容量限制** | 预约人数不能超过空间容量<br>会议室: 容量10-20人<br>会场: 容量50-500人<br>VIP餐厅: 容量6-20人 |
| **预约审批规则** | 会议室: 内部员工无需审批，外部用户需审批<br>会场: 必须审批<br>大屏: 内容审核后审批<br>展位: 方案审批<br>VIP餐厅: 内部需主管审批，外部预付确认 |
| **预约取消规则** | 取消时间限制: 根据定金规则确定<br>取消流程: 取消申请 → 审批（如需） → 定金退还（如有） → 取消确认 |

### 4.2 定金规则

| 空间类型 | 定金比例 | 退还规则 |
|---------|---------|---------|
| 会议室 | 无定金（内部月结） | 取消无需定金处理 |
| 会场 | 30%定金 | 提前7天取消: 100%退回<br>提前3-7天: 50%退回<br>提前1-3天: 不退<br>使用前24小时: 不退 |
| 大屏 | 无定金（预付全款） | 取消需审核退款 |
| 展位 | 50%定金 | 提前7天取消: 100%退回<br>提前3-7天: 50%退回<br>提前1-3天: 不退<br>使用前24小时: 不退 |
| VIP餐厅 | 30%定金（外部） | 提前24小时取消: 100%退回<br>当天取消: 不退<br>用餐后取消: 不退 |

### 4.3 结算规则

| 空间类型 | 结算方式 | 结算时机 |
|---------|---------|---------|
| 会议室 | 内部月结 | 月底统一结算 |
| 会场 | 预付定金+尾款 | 活动前付清尾款 |
| 大屏 | 预付全款 | 预约时支付 |
| 展位 | 预付定金+尾款 | 展示前付清尾款 |
| VIP餐厅 | 内部餐后结算，外部预付定金+尾款 | 用餐后结算（内部），用餐前付清（外部） |
